Jenkins ↔ Microsoft Teams 알림 설정

1. Microsoft Teams 에서의 설정 방법

  1. Teams 에서의 설정 방법

    • 채널 생성

    • Webhook 설정

      • 앱 > Incoming Webhook 검색 > 팀에 추가 > 팀 또는 채널 선택 > 커넥터 설정

    • 커넥터를 통한 Jenkins 설치

      • 채널 > 채널 관리 > 커넥터 편집 > Jenkins 검색 > add

    • 커넥터를 통한 Jenkins 환경 설정

      • 채널 > 채널 관리 > 커넥터 편집 > Jenkins 검색 > Configure 선택 > 이름 적기 > Create

    • Webhook URL 발급

      • https://diquest01.webhook.office.com/webhookb2/e4ea9ac5-2919-443a-a873-08b00866dec9@eedee5a3-6e77-4509-9cd4-ba89a5dc6a8e/JenkinsCI/872171e15a614a93b284297bdf1186fc/2fd28cbf-015b-48e7-9578-17c004f58d4a

  • Jenkins에서 설정

    • Office 365 Connector 플러그인 설치

      • Dashboard > Jenkins 관리 > Plugins > Available plugins 탭 클릭 > Office 365 Connector 검색 > 체크 후 install 버튼 클릭 > restart

    • 프로젝트에서 Office 365 환경 설정 :

      • Dashboard > 프로젝트 > Configuration > Scan Multibranch Pipeline Triggers

    • Jenkinsfile 에서 Microsoft Teams 알림 설정

Email 알림 설정

  1. Email Extension Plugin 설치 → 설치 시, 기본 패키지를 선택했으면 이미 설치되어져 있음

    • Dashboard > Jenkins 관리 > Plugins > Available plugins

  2. 이메일 설정 (Dashboard > Jenkins 관리 > System) : 여기서는 Microsoft outlook을 대상으로 함

    • Jenkins Location

      • System Admin e-mail address : SMTP 서버 Admin 계정 (Jenkins 어드민이 아님)

    • Extended E-mail Notification

      • SMTP Server : smtp-mail.outlook.com
      • SMTP Port : 587
      • Credentials
        • add > SMTP 의 아이디와 비밀번호 입력
        • Use TLS 체크
      • Default user e-mail suffix : @diquest.com
      • Default Recipients : shchoice@diquest.com

      • 테스트 이메일 전송하고 확인해보기

3.1. Jenkinsfile에서 이메일 알림 추가

3.2. Freestyle Project로 설정 시

  • Job 설정에서 이메일 알림 추가

    • Editable Email Notification 선택

    • Project Recipient List (이메일 수신자 설정)

    • Triggers (이메일 트리거 설정) > Add Triggers

      • Failure: 빌드가 실패할 때
      • Success: 빌드가 성공할 때
      • Always: 빌드가 성공하든 실패하든 항상 이메일을 전송
      • Failure: 빌드가 실패했을 때 이메일을 전송
      • Success: 빌드가 성공했을 때 이메일을 전송
      • Unstable: 빌드가 불안정한 경우(예: 일부 테스트 실패) 이메일을 전송
      • First Failure: 빌드가 처음 실패했을 때만 이메일을 전
      • Fixed: 이전 빌드가 실패했지만, 이번 빌드가 성공한 경우 이메일을 전송
