FastAPI - Deploy

FastAPIмҷҖ ASGI м„ңлІ„мқё uvicorn + WSGI м„ңлІ„мқё gunicorn мқ„ нҶөн•ң мӣ№ м„ңл№„мҠӨ кө¬нҳ„ л°Ҹ л°°нҸ¬лҘј н•ҳл©ҙм„ң к¶ҒкёҲн–ҲлҚҳ лӮҙмҡ©л“Өм—җ лҢҖн•ң м •лҰ¬н•ң лӮҙмҡ©мһ…лӢҲлӢӨ.

FastAPI мҷҖ uvicornмқ„ нҶөн•ҙ REST API м„ңлІ„лҘј кө¬нҳ„н•ҳкі , нҳ„мһ¬ мғҒнҷ©м—җ мөңм Ғнҷ”лҗң ProductлЎң л°°нҸ¬н•ҳкё° мң„н•ҙ мһҗлЈҢмЎ°мӮ¬лҘј н•ҳлҚҳ мӨ‘м—җ gunicorn + uvicorn мқ„ нҶөн•ң м„ңл№„мҠӨлҘј м•ҢкІҢ лҗҳм—ҲмҠөлӢҲлӢӨ. к·ёлһҳм„ң к°Ғк°Ғм—җ лҢҖн•ң к°ңл…җкіј мӣ№ м„ңл№„мҠӨлҘј м–ҙл–»кІҢ мөңм Ғнҷ”н•ҳм—¬ л°°нҸ¬лҘј н•  мҲҳ мһҲлҠ”м§Җ м •лҰ¬н•ҳмҳҖмҠөлӢҲлӢӨ.

ASGI м„ңлІ„мҷҖ WSGI м„ңлІ„лһҖ?

ASGI м„ңлІ„мҷҖ WSGI м„ңлІ„к°Җ л¬ҙм—Үмқём§Җ м•Ңкё° мң„н•ҙм„ң CGI л°Ҹ WASмҷҖ н•Ёк»ҳ м•Ңм•„ліҙкІ мҠөлӢҲлӢӨ.

  • CGI(Common Gateway Interface)

    • мӣ№ м„ңлІ„мҷҖ мҷёл¶Җ н”„лЎңк·ёлһЁ к°„мқҳ лҚ°мқҙн„° көҗнҷҳмқ„ мң„н•ң мқён„°нҺҳмқҙмҠӨ к·ңм•Ҫ (мқјмў…мқҳ мӣ№ м„ңлІ„ кё°мҲ лЎң ліј мҲҳ мһҲмқҢ)
      • мҰү, мӣ№ м„ңлІ„мҷҖ мҷёл¶Җ н”„лЎңк·ёлһЁ к°„мқҳ мғҒнҳёмһ‘мҡ©мқ„ мң„н•ҙ н‘ңмӨҖнҷ”лҗң л°©мӢқмқ„ м ңкіө
    • nginx, apache мӣ№ м„ңлІ„м—җм„ң мҡ°лҰ¬к°Җ мһ‘м„ұн•ң н”„лЎңк·ёлһЁмқ„ мӢӨн–үн•ҳкё° мң„н•ң мқён„°нҺҳмқҙмҠӨ
    • н•ҳм§Җл§Ң нҳ„мһ¬лҠ” лҢҖл¶Җл¶„ мӣ№ н”„л Ҳмһ„мӣҢнҒ¬лҘј нҶөн•ҙ лҸҷм Ғмқё мӣ№ нҺҳмқҙм§ҖлҘј мғқм„ұ(CGIлҠ” кіјкұ°м—җ мЈјлЎң мӮ¬мҡ©)
    • мӣ№ нҺҳмқҙм§Җм—җ лҸҷм Ғмқё мҪҳн…җмё лҘј м ңкіөн•ҳкё° мң„н•ҙ мӣ№ м„ңлІ„к°Җ нҒҙлқјмқҙм–ёнҠё мҡ”мІӯм—җ л”°лқј мҷёл¶Җ н”„лЎңк·ёлһЁмқ„ мӢӨн–үн•ҳкі  к·ё кІ°кіјлҘј л°ҳнҷҳн•ҳлҠ” л°©мӢқмңјлЎң лҸҷмһ‘
    • мҡ”мІӯ вҶ’ мӣ№м„ңлІ„(nginx, apache) вҶ’ CGI вҶ’ CGIм—җ мһ‘м„ұн•ң н”„лЎңк·ёлһЁ мӢӨн–ү(н”„лЎңм„ёмҠӨ мғқм„ұ)
    • н”„лЎңм„ёмҠӨлҘј мғқм„ұн•ҳм—¬ мӢӨн–ү
      • мҰү fork, spawnлҘј нҶөн•ҙ н”„лЎңм„ёмҠӨлҘј мғқм„ұ
    • мҡ”мІӯм—җ л”°лқј н”„лЎңм„ёмҠӨлҘј мғқм„ұ
      • л§ҺмқҖ мҡ”мІӯмқҙ л“Өм–ҙмҳӨл©ҙ м„ңлІ„ л¶Җн•ҳк°Җ мҰқк°Җн•ҳкі  м„ұлҠҘ м Җн•ҳ(л©”лӘЁлҰ¬ л“ұ)лҘј мҙҲлһҳн•  мҲҳ мһҲмқҢ

  • WAS(Web Application Server)

    • лҢҖк·ңлӘЁ мӣ№ м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқ„ мҡҙмҳҒн•ҳлҠ”лҚ° л§Өмҡ° мң мҡ©

    • мӣ№ м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқ„ мӢӨн–үн•ҳкё° мң„н•ң лҜёл“ӨмӣЁм–ҙ н”Ңлһ«нҸјмңјлЎң, мӣ№ м„ңлІ„мҷҖ лҚ°мқҙн„°лІ мқҙмҠӨ м„ңлІ„ мӮ¬мқҙмқҳ мӨ‘к°„ кі„мёөм—җм„ң лҸҷмһ‘

      WASмқҳ кё°лҠҘ м°ёкі 

      • мӣ№ м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқ„ мӢӨн–үн•ҳкё° мң„н•ң лҹ°нғҖмһ„ нҷҳкІҪ м ңкіө
      • лӢӨм–‘н•ң н”„лЎңк·ёлһҳл°Қ м–ём–ҙмҷҖ н”„л Ҳмһ„мӣҢнҒ¬лҘј м§Җмӣҗ
      • м„ём…ҳ кҙҖлҰ¬, нҠёлһңмһӯм…ҳ кҙҖлҰ¬, ліҙм•Ҳ л“ұмқҳ кё°лҠҘ м ңкіө
      • лЎңл“ң л°ёлҹ°мӢұ, нҒҙлҹ¬мҠӨн„°л§Ғ, мәҗмӢұ л“ұмқҳ кё°лҠҘ м ңкіө
      • лӘЁлӢҲн„°л§Ғ, лЎңк№…, м„ұлҠҘ мёЎм • л“ұмқҳ кё°лҠҘ м ңкіө

    • мҡ”мІӯ вҶ’ мӣ№м„ңлІ„(nginx, apache) вҶ’ WAS(Tomcat) вҶ’ WASм—җ мһ‘м„ұн•ң н”„лЎңк·ёлһЁ мӢӨн–ү(м“°л Ҳл“ң мӢӨн–ү)

      • (н•ҳм§Җл§Ң мҡ”мҰҳм—җлҠ” мөңк·јм—җлҠ” WAS мһҗмІҙк°Җ мӣ№ м„ңлІ„ кё°лҠҘмқ„ нҸ¬н•Ён•ҙ лӢЁмқј м„ңлІ„лЎң мӣ№ м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқ„ кө¬м¶•н•ҳлҠ” кІғмқҙ мқјл°ҳм Ғ)

    • CGIлҠ” н”„лЎңм„ёмҠӨлҘј мғқм„ұн•ҳм—¬ мӢӨн–үн•ҳм§Җл§Ң WASлҠ” м“°л Ҳл“ңлҘј нҶөн•ҙ мӢӨн–үлҗЁ

    • Django, Flask, FastAPIлҠ” CGI, WASмҷҖ к°ҷмқҖ м„ңлІ„к°Җ мЎҙмһ¬н•ҙм•ј н•Ё

      • FastAPI : ASGI м„ңлІ„к°Җ н•„мҡ”н•Ё
      • Flask : WSGI м„ңлІ„к°Җ н•„мҡ”
      • Django : WSGI, ASGI лӘЁл‘җ м§Җмӣҗ

      Deploying to Production

      Django, being a web framework, needs a web server in order to operate. And since most web servers donвҖҷt natively speak Python, we need an interface to make that communication happen. Django currently supports two interfaces: WSGI and ASGI.

      вҖ» https://flask.palletsprojects.com/en/2.2.x/deploying/

  • WSGI(Web Server Gateway Interface)

    • Python мӣ№ м–ҙн”ҢлҰ¬мјҖмқҙм…ҳ к°ңл°ңмқҳ н‘ңмӨҖ мқён„°нҺҳмқҙмҠӨ

    • Python мӣ№ м–ҙн”ҢлҰ¬мјҖмқҙм…ҳкіј мӣ№ м„ңлІ„ к°„мқҳ лҚ°мқҙн„°лҘј мЈјкі  л°ӣкё° мң„н•ң мқён„°нҺҳмқҙмҠӨ к·ңкІ©

    • мӣ№ м„ңлІ„мҷҖ мӣ№ н”„л Ҳмһ„мӣҢнҒ¬ мӮ¬мқҙм—җ мЎҙмһ¬ вҶ’ WASлҘј лҢҖмІҙн•ңлӢӨкі  ліҙл©ҙ лҗЁ

    • CGIлҠ” н”„лЎңм„ёмҠӨлҘј мғқм„ұн•ҳм—¬ мҡ°лҰ¬к°Җ мһ‘м„ұн•ң мҪ”л“ңлҘј мӢӨн–үн•ҳлҠ” л°ҳл©ҙ WSGIлҠ” мҡ”мІӯм—җ лҢҖн•ҙ мҡ°лҰ¬к°Җ мһ‘м„ұн•ң мҪ”л“ңлҘј мҪңл°ұн•ҳм—¬ нҳём¶ңн•ҳлҠ” м—ӯн• 

      • мҰү, WSGIлҠ” мӣ№ н”„л Ҳмһ„мӣҢнҒ¬мқҳ мҪ”л“ңлҘј нҳём¶ңн•ҙ мЈјлҠ” м—ӯн• 

    • лӢЁм җмңјлЎңлҠ” л№„лҸҷкё° мІҳлҰ¬к°Җ нһҳл“ӨмқҢ

      • WebsocketмқҙлӮҳ кёҙ HTTP мҡ”мІӯмқ„ мІҳлҰ¬н•ҳкё°м—җ м Ғн•©н•ҳм§Җ м•ҠмқҢ(лӢЁмқј лҸҷкё° нҳём¶ң л°©мӢқмңјлЎң мІҳлҰ¬лҗҳм–ҙ мҳӨлһң мӢңк°„ м—°кІ°мқ„ мң м§Җ)
      • WSGIлҠ” мҡ”мІӯм—җ лҢҖн•ң мҪңл°ұмқҙ лҸҷкё°м Ғмқё нҳ•нғңлҘј к°ҖмЎҢкё° л•Ңл¬ё

  • ASGI(Asynchronous Server Gateway Interface)

    • Python л№„лҸҷкё° мӣ№ м–ҙн”ҢлҰ¬мјҖмқҙм…ҳкіј мӣ№ м„ңлІ„ к°„мқҳ мқён„°нҺҳмқҙмҠӨ к·ңкІ©
    • WebSocketкіј к°ҷмқҖ мӢӨмӢңк°„ нҶөмӢ мқ„ м§Җмӣҗ(лӢЁмқј л№„лҸҷкё° нҳём¶ңмқҙ к°ҖлҠҘн•ҳлҸ„лЎқ м„Өкі„лҗЁ)
    • нҒҙлқјмқҙм–ёнҠёлЎңл¶Җн„° м—¬лҹ¬ мқҙлІӨнҠёлҘј мЈјкі л°ӣмқ„ мҲҳ мһҲмңјл©°, л°ұк·ёлқјмҡҙл“ң мҪ”лЈЁнӢҙмқ„ мӢӨн–үн•  мҲҳ мһҲкІҢ лҗЁ
    • SGIлҠ” WSGIм—җ л№„н•ҙ лҚ” лҶ’мқҖ м„ұлҠҘкіј м•Ҳм •м„ұ, к·ёлҰ¬кі  кё°лҠҘмқ„ м ңкіө
    • л№„лҸҷкё° л°©мӢқмңјлЎң лҸҷмһ‘н•ҳлҜҖлЎң л©ҖнӢ° мҠӨл Ҳл“ң нҷҳкІҪм—җм„ң л°ңмғқн•  мҲҳ мһҲлҠ” л¬ём ңлҘј н•ҙкІ°
    • Pythonмқҳ asyncio лқјмқҙлёҢлҹ¬лҰ¬лҘј кё°л°ҳмңјлЎң н•ҳм—¬ лӘЁл“Ҳ кҙҖлҰ¬к°Җ лҚ”мҡұ к°„нҺё

uvicornкіј gunicorn мқҙлһҖ

  • Uvicorn
    • Python ASGI(Asynchronous Server Gateway Interface) мӣ№ м„ңлІ„ (Python л№„лҸҷкё° мӣ№ м• н”ҢлҰ¬мјҖмқҙм…ҳмқ„ мІҳлҰ¬н•ҳлҠ” лҚ° мӮ¬мҡ©)
    • кё°мЎҙмқҳ мӣ№ м„ңлІ„мҷҖ лӢ¬лҰ¬ л№„лҸҷкё° л°©мӢқмңјлЎң лҸҷмһ‘н•ҳм—¬ лҶ’мқҖ м„ұлҠҘмқ„ м ңкіө
      • л№„лҸҷкё° л°©мӢқмңјлЎң лҸҷмһ‘н•ҳкё° л•Ңл¬ём—җ, лё”лЎңнӮ№ м—°мӮ° м—ҶмқҙлҸ„ л№ лҘё мқ‘лӢө мҶҚлҸ„лҘј м ңкіөн•  мҲҳ мһҲ
    • asyncio лқјмқҙлёҢлҹ¬лҰ¬лҘј мӮ¬мҡ©н•ҳм—¬ л№„лҸҷкё° л°©мӢқмңјлЎң лҸҷмһ‘
      • cypthon мңјлЎң кө¬нҳ„н•ң uvloop лҘј мӮ¬мҡ©н•ҳм—¬ мқҙлІӨнҠё лЈЁн”„лҘј кө¬нҳ„(л№„лҸҷкё° м„ңлІ„лҘј кө¬нҳ„)
      • л„ӨнҠёмӣҢнҒ¬ I/O м—җ лҢҖн•ң мҪңл°ұ мң„мЈјлЎң кҙҖлҰ¬
    • UvicornмқҖ лӢӨмӨ‘ н”„лЎңм„ёмҠӨмҷҖ лӢӨмӨ‘ мҠӨл Ҳл“ңлҘј м§Җмӣҗн•ҳм—¬ лҶ’мқҖ мІҳлҰ¬лҹүмқ„ м ңкіөн•ҳл©°, HTTPS, HTTP/2, WebSocket л“ұ лӢӨм–‘н•ң н”„лЎңнҶ мҪңмқ„ м§Җмӣҗ
    • Python 3.6 мқҙмғҒмқҳ лІ„м „м—җм„ң мӮ¬мҡ©н•  мҲҳ мһҲмқҢ
  • Gunicorn
    • Python мӣ№ м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқ„ мң„н•ң WSGI(Web Server Gateway Interface) мӣ№ м„ңлІ„
    • GunicornмқҖ лӢӨмӨ‘ н”„лЎңм„ёмҠӨлҘј мӮ¬мҡ©н•ҳм—¬ Python мӣ№ м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқ„ мІҳлҰ¬н•ҳл©°, лӢӨмӨ‘ мҪ”м–ҙ мӢңмҠӨн…ңм—җм„ң лҶ’мқҖ м„ұлҠҘмқ„ м ңкіө
    • н”„лЎңм„ёмҠӨ кҙҖлҰ¬мһҗ м—ӯн• мқ„ мҲҳн–ү

FastAPIмҷҖ uvicornкіјмқҳ кҙҖкі„

  • FastAPIлҠ” ASGI мқён„°нҺҳмқҙмҠӨк°Җ кө¬нҳ„лҗң starletteлҘј м¶”мғҒнҷ”н•ҳм—¬ кө¬нҳ„лҗң мӣ№ н”„л Ҳмһ„мӣҢнҒ¬

    [мӣ№ н”„л Ҳмһ„мӣҢнҒ¬]FastAPI - Starlette - [ASGI мӣ№ м„ңлІ„]Uvicorn - uvloop - libuv

  • FastAPIлҘј мӮ¬мҡ©н•ҳм—¬ Python мӣ№ м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқ„ к°ңл°ңн•ҳкі , ASGI м„ңлІ„ мӨ‘ н•ҳлӮҳмқё Uvicornмқ„ мӮ¬мҡ©н•ҳм—¬ мӢӨн–ү

FastAPIмҷҖ uvicorn + gunicornкіјмқҳ кҙҖкі„

  • gunicorn + uvicorn мқ„ мЎ°н•©н•ҳм—¬ мӮ¬мҡ©н•  мҲҳ мһҲмңјл©° ASGIмҷҖ WSGIмқҳ нҠ№м§•мңјлЎң лӢӨмқҢкіј к°ҷмқҖ нҡЁкіјлҘј м–»мқ„ мҲҳ мһҲмқҢ

    • л№„лҸҷкё° мІҳлҰ¬
      • Uvicornм—җм„ң л№„лҸҷкё° мІҳлҰ¬к°Җ н•„мҡ”н•ң л¶Җл¶„мқ„ GunicornмңјлЎң м—°кІ°н•ҳм—¬ л№„лҸҷкё° мІҳлҰ¬лҘј м§Җмӣҗ
    • м•Ҳм •м„ұ:
      • UvicornмқҖ л№„лҸҷкё° мІҳлҰ¬ л°©мӢқмңјлЎң лҸҷмһ‘н•ҳкё° л•Ңл¬ём—җ, мІҳлҰ¬лҹүмқҙ лҶ’м•„м§Ҳ л•ҢлҸ„ м•Ҳм •м ҒмңјлЎң лҸҷмһ‘н•  мҲҳ мһҲмқҢ
      • GunicornмқҖ лӢӨмӨ‘ н”„лЎңм„ёмҠӨ л°©мӢқмңјлЎң лҸҷмһ‘н•ҳкё° л•Ңл¬ём—җ, м•Ҳм •м„ұмқ„ ліҙмһҘн•  мҲҳ мһҲмқҢ
      • л‘җ мӣ№ м„ңлІ„лҘј н•Ёк»ҳ мӮ¬мҡ©н•ҳл©ҙ, м•Ҳм •м„ұмқ„ ліҙмһҘн•ҳл©ҙм„ңлҸ„ лҶ’мқҖ мІҳлҰ¬лҹүмқ„ м ңкіөн•  мҲҳ мһҲмқҢ

  • gunicorn + uvicorn лҘј н•Ёк»ҳ мӮ¬мҡ©н–Ҳмқ„ л•Ңмқҳ м•„нӮӨн…ҚмІҳ
