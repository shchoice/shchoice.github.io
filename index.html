UML & Class Diagram

UML мқҙлһҖ

  • UML(Unified Modeling Language)мқҖ мӢңмҠӨн…ңмқҳ кө¬мЎ°мҷҖ м„Өкі„лҘј мӢңк°Ғнҷ”н•ҳлҠ” лҚ° мӮ¬мҡ©лҗҳлҠ” н‘ңмӨҖнҷ”лҗң лӘЁлҚёл§Ғ м–ём–ҙ

  • м¶”мғҒм Ғмқё к°ңл…җмқ„ мӢңк°Ғм Ғ нҳ•нғңлЎң н‘ңнҳ„н•ЁмңјлЎңмҚЁ ліөмһЎн•ң мӢңмҠӨн…ңмқҳ м„Өкі„мҷҖ л¶„м„қмқ„ мҡ©мқҙн•ҳкІҢ н•ҳлҠ” мӨ‘мҡ”н•ң лҸ„кө¬

  • лӢӨмқҢкіј к°ҷмқҖ мў…лҘҳк°Җ мһҲмқҢ

    1. нҒҙлһҳмҠӨ лӢӨмқҙм–ҙк·ёлһЁ(Class Diagrams): мӢңмҠӨн…ң лӮҙмқҳ нҒҙлһҳмҠӨл“Өкіј к·ёл“Ө мӮ¬мқҙмқҳ кҙҖкі„лҘј ліҙм—¬мӨҢ
    2. мӮ¬мҡ© мӮ¬лЎҖ лӢӨмқҙм–ҙк·ёлһЁ(Use Case Diagrams): мӢңмҠӨн…ңмқҙ мӮ¬мҡ©мһҗмҷҖ мғҒнҳё мһ‘мҡ©н•ҳлҠ” л°©мӢқмқ„ м„ӨлӘ…
    3. мӢңнҖҖмҠӨ лӢӨмқҙм–ҙк·ёлһЁ(Sequence Diagrams): к°қмІҙ к°„мқҳ мғҒнҳё мһ‘мҡ©кіј мӢңк°„м—җ л”°лҘё мқҙл“Өмқҳ кҙҖкі„лҘј лӮҳнғҖлғ„
    4. мғҒнғң лӢӨмқҙм–ҙк·ёлһЁ(State Diagrams): к°қмІҙмқҳ мғҒнғң ліҖнҷ”лҘј ліҙм—¬мӨҢ
    5. нҷңлҸҷ лӢӨмқҙм–ҙк·ёлһЁ(Activity Diagrams): мӣҢнҒ¬н”ҢлЎңмҡ°лӮҳ л№„мҰҲлӢҲмҠӨ н”„лЎңм„ёмҠӨмқҳ нқҗлҰ„мқ„ лӮҳнғҖлғ„
    6. кө¬м„ұ лӢӨмқҙм–ҙк·ёлһЁ(Component Diagrams): мӢңмҠӨн…ңмқҳ л¬јлҰ¬м Ғ кө¬м„ұ мҡ”мҶҢлҘј лӮҳнғҖлғ„
    7. л°°м№ҳ лӢӨмқҙм–ҙк·ёлһЁ(Deployment Diagrams): мӢңмҠӨн…ңмқҳ л¬јлҰ¬м Ғ л°°м№ҳмҷҖ мқён”„лқјлҘј лӮҳнғҖлғ„

    • кө¬мЎ° лӢӨмқҙм–ҙк·ёлһЁ(Structure Diagram)
      • мӢңмҠӨн…ңмқҳ м •м Ғмқё мёЎл©ҙ, мҰү мӢңмҠӨн…ңмқҳ л¬јлҰ¬м Ғ кө¬м„ұ мҡ”мҶҢмҷҖ к·ёл“Ө к°„мқҳ кҙҖкі„лҘј лӮҳнғҖлғ„
      • мӢңмҠӨн…ңмқҳ кө¬мЎ°лҘј ліҙм—¬мЈјл©°, мЈјмҡ” кө¬м„ұ мҡ”мҶҢл“Өмқҙ м–ҙл–»кІҢ м„ңлЎң м—°кІ°лҗҳм–ҙ мһҲлҠ”м§ҖлҘј м„ӨлӘ…
      • мў…лҘҳ
        1. нҒҙлһҳмҠӨ лӢӨмқҙм–ҙк·ёлһЁ(Class Diagram): мӢңмҠӨн…ңмқҳ нҒҙлһҳмҠӨл“Өкіј к·ёл“Ө мӮ¬мқҙмқҳ кҙҖкі„лҘј ліҙм—¬мӨҢ
        2. к°қмІҙ лӢӨмқҙм–ҙк·ёлһЁ(Object Diagram): нҠ№м • мӢңм җм—җм„ң нҒҙлһҳмҠӨ лӢӨмқҙм–ҙк·ёлһЁмқҳ мқёмҠӨн„ҙмҠӨлҘј ліҙм—¬мӨҢ
        3. нҢЁнӮӨм§Җ лӢӨмқҙм–ҙк·ёлһЁ(Package Diagram): мӢңмҠӨн…ңмқҳ нҢЁнӮӨм§Җл“Өкіј к·ёл“Ө к°„мқҳ мқҳмЎҙм„ұмқ„ лӮҳнғҖлғ„
        4. кө¬м„ұ лӢӨмқҙм–ҙк·ёлһЁ(Component Diagram): мӢңмҠӨн…ңмқҳ мҶҢн”„нҠёмӣЁм–ҙ кө¬м„ұ мҡ”мҶҢл“Өмқ„ ліҙм—¬мӨҢ.
        5. л°°м№ҳ лӢӨмқҙм–ҙк·ёлһЁ(Deployment Diagram): мӢңмҠӨн…ңмқҳ л¬јлҰ¬м Ғ кө¬мЎ°мҷҖ мҶҢн”„нҠёмӣЁм–ҙ кө¬м„ұ мҡ”мҶҢл“Өмқҙ л°°м№ҳлҗҳлҠ” л°©лІ•мқ„ лӮҳнғҖлғ„
    • н–үмң„ лӢӨмқҙм–ҙк·ёлһЁ(Behavior Diagram)
      • мӢңмҠӨн…ңмқҳ лҸҷм Ғмқё мёЎл©ҙ, мӢңмҠӨн…ңмқҙ м–ҙл–»кІҢ лҸҷмһ‘н•ҳкі , кө¬м„ұ мҡ”мҶҢл“Өмқҙ мӢңк°„м—җ л”°лқј м–ҙл–»кІҢ мғҒнҳё мһ‘мҡ©н•ҳлҠ”м§ҖлҘј ліҙм—¬мӨҢ
      • мӢңмҠӨн…ңмқҳ н–үлҸҷ, н”„лЎңм„ёмҠӨ, м•Ңкі лҰ¬мҰҳ л“ұмқ„ мӢңк°Ғнҷ”
      • мў…лҘҳ
        1. мӮ¬мҡ© мӮ¬лЎҖ лӢӨмқҙм–ҙк·ёлһЁ(Use Case Diagram): мӢңмҠӨн…ңмқҳ кё°лҠҘкіј мҷёл¶Җ мҡ”мҶҢ(мӮ¬мҡ©мһҗ, мҷёл¶Җ мӢңмҠӨн…ң л“ұ)мҷҖмқҳ мғҒнҳё мһ‘мҡ©мқ„ лӮҳнғҖлғ„
        2. мӢңнҖҖмҠӨ лӢӨмқҙм–ҙк·ёлһЁ(Sequence Diagram): к°қмІҙ к°„мқҳ мғҒнҳё мһ‘мҡ©кіј мқҙл“Өмқҙ л©”мӢңм§ҖлҘј көҗнҷҳн•ҳлҠ” мҲңм„ңлҘј ліҙм—¬мӨҢ
        3. нҷңлҸҷ лӢӨмқҙм–ҙк·ёлһЁ(Activity Diagram): мӢңмҠӨн…ңмқҳ н”„лЎңм„ёмҠӨ нқҗлҰ„мқҙлӮҳ мӣҢнҒ¬н”ҢлЎңмҡ°лҘј лӮҳнғҖлғ„
        4. мғҒнғң лӢӨмқҙм–ҙк·ёлһЁ(State Diagram): к°қмІҙмқҳ мғҒнғң ліҖнҷ”мҷҖ к·ём—җ л”°лҘё мқҙлІӨнҠё мІҳлҰ¬лҘј лӮҳнғҖлғ„.
        5. м»Өл®ӨлӢҲмјҖмқҙм…ҳ лӢӨмқҙм–ҙк·ёлһЁ(Communication Diagram): мӢңнҖҖмҠӨ лӢӨмқҙм–ҙк·ёлһЁкіј мң мӮ¬н•ҳм§Җл§Ң, к°қмІҙ к°„мқҳ м—°кІ°кіј л©”мӢңм§Җ нқҗлҰ„м—җ лҚ” м§‘мӨ‘

  • Given dataset

    • $D={x^i, y^i}_{i=1}^{N}$ // рқ‘Ҙ ~ рқ‘ғ(рқ‘Ҙ) y ~ рқ‘ғ(рқ‘Ҙ) рқ‘Ҙм—җм„ң yлЎң мҰү н•ңкёҖмқ„ мҳҒм–ҙлЎң лІҲм—ӯ.

      $x^i = {x_1^i, \ldots, x_m^i} \text{ } y^i = {y_0^i, y_1^i, \ldots, y_n^i}$

      where $y_0=$, $y_n=$

  • Find parameter that maximize likelihood,

    • $\hat{\theta} = \text{argmax}{\theta \in \Theta} \sum{i=1}^{N} \log P(y^i|x^{i}; \theta)$ // лҚ°мқҙн„°м—җ лҢҖн•ҙ log-likelihoodлҘј мөңлҢҖнҷ”н•ҳлҠ” нҢҢлқјлҜён„°лҘј м°ҫмһҗ

      $= \text{argmax}{\theta \in \Theta} \sum{i=1}^{N}\sum_{j=1}^{n} \log P(y_{j}^{i}|x^i, y_{<j}^i; \theta)$ // by chain-rule

  • Minimize loss function by upgrading parameter with gradient descent.

    • $L(\theta) = -\sum_{i=1}^{N}\sum_{j=1}^{n} \log P(y_{j}^{i},|x^i,y_{<j}^i; \theta)$

      $\theta \leftarrow \theta - \eta\nabla_\theta L(\theta)$

      $\log P(x_t | x_{<t}; \theta) = x_t^T \cdot \log f_\theta(x_{t-1}, h_{t-1})$ // softmax layerлҘј нҶөкіјн•ҳлҠ” кІҪмҡ°,
      // $x_t^T$ : GT (One-hot vector, мӢӨм ң мҳҲмёЎ лӢЁм–ҙ)
      // $\log f_\theta(x_{t-1}, h_{t-1})$ : LM(softmax layerм—җ log м”Ңмҡҙ нҷ•лҘ к°’)

      $\text{where } x_t \text{ is one- hot vector, and }f_Оё \text{ is model with parameter Оё.}$

    • $w = \text{softmax}(h_{t}^{dec} \cdot W_{a} \cdot h_{1:m}^{enc^T}))$

      • $h_{t}^{dec} \cdot W_{a}$ : Q
      • $h_{1:m}^{enc^T}$ : $K^T$
      • |Q|=(bs,1,hs), |K|=(bs,hs,m), |QвҲҷK|=(bs,1,m)

      $c=w \cdot h_{1:m}^{enc},$

      $\text{where } c \in \mathbb{R}^{batchsize \times 1 \times hiddensize}$ is a context vector, and $W_a \in \mathbb{R}^{batchsize \times hiddensize}$

    • $\tilde{h}t^{dec} = \tanh([(h_t^{dec}; c)] \cdot W{concat})$

      • $[(h_t^{dec}; c)]$ : $в„Һ_рқ‘Ў^{рқ‘‘рқ‘’рқ‘җ}$ кіј cлҘј concatн•ҳл©ҙ (bs, 1, hs*2)

      $\hat{y}_t = \text{softmax}(\tilde{h}t^{dec} \cdot W{gen})$

      • $|\hat{y}_t|$=(bs,1,hs)*(hs, |V|) = (bs,1,|V|)

      $\text{where } W_{concat} \in \mathbb{R}^{(2 \times hiddensize) \times hiddensize} \text{ and } W_{gen} \in \mathbb{R}^{hiddensize \times |V|}$

    • $L(\theta) = -\sum_{i=1}^{N}\sum_{t=1}^{n} \log P(y_{j}^{i},|x^i,y_{<t}^i; \theta)$
      $\log P(x_t | x_{<t}; \theta) = y_t^{iT} \cdot \log{\hat{y}{t}^i})$ // softmax layerлҘј нҶөкіјн•ҳлҠ” кІҪмҡ°,
      // $|y_t^{iT}|$$y_t^{iT}$ : (bs,1,|v|)
      // $|\log{\hat{y}      {t}^i}|$$\log{\hat{y}_{t}^i}$ : (bs,|v|,1)
      $\text{where } x_t \text{ is one- hot vector, and }f_Оё \text{ is model with parameter Оё.}$
      $\theta \leftarrow \theta - \eta\nabla_\theta L(\theta)$

  • Given dataset

    • $D={x^i, y^i}_{i=1}^{N}$ // рқ‘Ҙ ~ рқ‘ғ(рқ‘Ҙ) y ~ рқ‘ғ(рқ‘Ҙ) рқ‘Ҙм—җм„ң yлЎң мҰү н•ңкёҖмқ„ мҳҒм–ҙлЎң лІҲм—ӯ.

      $x^i = {x_1^i, \ldots, x_m^i} \text{ } y^i = {y_0^i, y_1^i, \ldots, y_n^i}$

      where $y_0=$, $y_n=$

  • Find parameter that maximize likelihood,

    • $\hat{\theta} = \text{argmax}{\theta \in \Theta} \sum{i=1}^{N} \log P(y^i|x^{i}; \theta)$ // лҚ°мқҙн„°м—җ лҢҖн•ҙ log-likelihoodлҘј мөңлҢҖнҷ”н•ҳлҠ” нҢҢлқјлҜён„°лҘј м°ҫмһҗ

      $= \text{argmax}{\theta \in \Theta} \sum{i=1}^{N}\sum_{j=1}^{n} \log P(y_{j}^{i}|x^i, y_{<j}^i; \theta)$ // by chain-rule

  • Minimize loss function by upgrading parameter with gradient descent.

    • $L(\theta) = -\sum_{i=1}^{N}\sum_{j=1}^{n} \log P(y_{j}^{i},|x^i,y_{<j}^i; \theta)$

      $\theta \leftarrow \theta - \eta\nabla_\theta L(\theta)$

      $\log P(x_t | x_{<t}; \theta) = x_t^T \cdot \log f_\theta(x_{t-1}, h_{t-1})$ // softmax layerлҘј нҶөкіјн•ҳлҠ” кІҪмҡ°,
      // $x_t^T$ : GT (One-hot vector, мӢӨм ң мҳҲмёЎ лӢЁм–ҙ)
      // $\log f_\theta(x_{t-1}, h_{t-1})$ : LM(softmax layerм—җ log м”Ңмҡҙ нҷ•лҘ к°’)

      $\text{where } x_t \text{ is one- hot vector, and }f_Оё \text{ is model with parameter Оё.}$

  • Find parameter that maximize likelihood for given training corpus

      $\hat{\theta} = \text{argmax}{\theta \in \Theta} \sum{i=1}^{N} \log P(x_{1:n}^{i}; \theta)$ // лҚ°мқҙн„°м—җ лҢҖн•ҙ log-likelihoodлҘј мөңлҢҖнҷ”н•ҳлҠ” нҢҢлқјлҜён„°лҘј м°ҫмһҗ

      // by chain-rule
      $\text{where } x_{1:n} = {x_1, вҖҰ, x_n}$

  • Take a step of gradient descent to minimize negative log-likelihood

    • $L(\theta) = -\sum_{i=1}^{N}\sum_{j=1}^{n} \log P(x_{j}^{i},|P(x_{<j}^i; \theta)$

      $\theta \leftarrow \theta - \eta\nabla_\theta L(\theta)$
      $\log P(x_t | x_{<t}; \theta) = x_t^T \cdot \log f_\theta(x_{t-1}, h_{t-1})$ // softmax layerлҘј нҶөкіјн•ҳлҠ” кІҪмҡ°,
      // $x_t^T$ : GT (One-hot vector, мӢӨм ң мҳҲмёЎ лӢЁм–ҙ)

        $=\text{softmax}(h_t \cdot W)\text{, where } W \in \mathbb{R}^{hidden-size \times |V|}$
