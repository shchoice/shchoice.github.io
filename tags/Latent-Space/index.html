1мһҘ. Representation Learning - AutoEncoders

AutoEncoders

  • Overview

    • мқёмҪ”лҚ”(encoder)мҷҖ л””мҪ”лҚ”(decoder)лҘј нҶөн•ҙ м••м¶•кіј н•ҙм ңлҘј мӢӨн–ү
      • мқёмҪ”лҚ”лҠ” мһ…л Ҙ(рқ‘Ҙ)мқҳ м •ліҙлҘј мөңлҢҖн•ң ліҙмЎҙн•ҳлҸ„лЎқ мҶҗмӢӨ м••м¶•мқ„ мҲҳн–ү
      • л””мҪ”лҚ”лҠ” мӨ‘к°„ кІ°кіјл¬ј(рқ‘§)мқҳ м •ліҙлҘј мһ…л Ҙ(рқ‘Ҙ)кіј к°ҷм•„м§ҖлҸ„лЎқ м••м¶• н•ҙм ң(ліөмӣҗ)лҘј мҲҳн–ү
    • ліөмӣҗмқ„ м„ұкіөм ҒмңјлЎң н•ҳкё° мң„н•ҙ, autoencoderлҠ” нҠ№м§•(feature)мқ„ м¶”м¶ңн•ҳлҠ” л°©лІ•мқ„ мһҗлҸҷмңјлЎң н•ҷмҠө

  • Encoder

    • ліөмӣҗм—җ н•„мҡ”н•ң м •ліҙлҘј мӨ‘мӢ¬мңјлЎң мҶҗмӢӨ м••м¶•мқ„ мҲҳн–ү

    • н•„мҡ”м—ҶлҠ” м •ліҙ(л»”н•ң нҠ№м§•)лҠ” лІ„лҰҙ мҲҳлҸ„ мһҲмқҢ

      • мҳҲмӢң1) мқјл°ҳм Ғмқё мӮ¬лһҢмқҳ м–јкөҙмқ„ н•ҷмҠөн•  л•Ң: мӮ¬лһҢмқҳ м–јкөҙм—җм„ң лҲҲмқҖ 2к°ңмқҙлӢӨ л“ұ

      • мҳҲмӢң2)

  • Bottleneck(рқ’ӣ)

    • мһ…л Ҙ(рқ’ҷ)м—җ л№„н•ҙ мһ‘мқҖ м°ЁмӣҗмңјлЎң кө¬м„ұ
    • л”°лқјм„ң м •ліҙмқҳ м„ нғқкіј м••м¶•мқҙ л°ңмғқ, м°Ёмӣҗм—җ л”°лқј м••м¶•мқҳ м •лҸ„лҘј кІ°м •н•Ё
      • м§‘м—җ л¶Ҳмқҙ лӮҳм„ң нғҲм¶ңн•  л•Ң, л¬ҙм—Үмқ„ л“Өкі  лӮҳк°Ҳ кІғмқёк°Җ?
    • к·ёлҹ¬лҜҖлЎң рқ’ӣ лҠ” мһ…л Ҙ(рқ’ҷ)м—җ лҢҖн•ң feature vectorлқјкі  н•  мҲҳ мһҲлӢӨ.
    • м••м¶•мқҳ нҡЁмңЁмқҙ лҶ’м•„м•ј н•ҳлҜҖлЎң, мһ…л Ҙм—җ л№„н•ҙ dense vectorмқј кІғ

  • Decoder

    • м••м¶•лҗң мӨ‘к°„ кІ°кіјл¬ј(рқ’ҷ)мқ„ л°”нғ•мңјлЎң мөңлҢҖн•ң мһ…л Ҙ(рқ’ӣ)кіј л№„мҠ·н•ҳкІҢ ліөмӣҗ : $\hat{x}$
    • ліҙнҶө MSELoss лҘј нҶөн•ҙ мөңм Ғнҷ” мҲҳн–ү ($MSE=|\hat{x}-x|^2_2$)
    • л»”н•ң м •ліҙлҠ” мЈјм–ҙм§Җм§Җ м•ҠлҚ”лқјлҸ„ м–ҙм°Ён”ј м•Ң мҲҳ мһҲкё°м—җ ліөмӣҗ к°ҖлҠҘ

Hidden Representation

Motivation

  • мқёмҪ”лҚ”лЎңл¶Җн„° лӮҳмҳЁ мӨ‘к°„ кІ°кіјл¬ј(рқ’ӣ)мқҖ мһ…л Ҙ(рқ’ҷ)м—җ лҢҖн•ң feature vectorмқҙлӢӨ.

  • feature vectorмқҳ к°Ғ м°ЁмӣҗмқҖ м–ҙл–Ө мқҳлҜёлҘј лӮҙнҸ¬н•ҳкі  мһҲмқ„к№Ң?

    • мқёмҪ”лҚ”мқҳ кІ°кіјл¬ј рқ’ӣлҘј plot н•ҳмҳҖмқ„ л•Ң, л№„мҠ·н•ң мғҳн”Ңл“ӨмқҖ л№„мҠ·н•ң кіім—җ мң„м№ҳн•Ёмқ„ нҷ•мқё

    • мқҙ plotмқҙ лҝҢл Өм§„ кіөк°„мқ„ hidden(latent) spaceлқјкі  л¶ҖлҰ„(мһ мһ¬кіөк°„, feature vectorк°Җ мң„м№ҳн•ҳлҠ” кіі)

      • Input spaceмқҳ MNIST мғҳн”Ңмқҙ latent spaceм—җ embedding лҗң кІғ

Mapping to Hidden(Latent) Space

  • к°Ғ л Ҳмқҙм–ҙмқҳ кІ°кіјл¬јмқ„ hidden vector лқјкі  л¶ҖлҰ„
  • лӘЁл‘җ feature vectorлқјкі  ліј мҲҳ мһҲмқҢ

Hidden(Latent) Representation

  • Tabular dataмқҳ feature vectorмҷҖ лӢ¬лҰ¬, hidden vectorлҠ” н•ҙм„қмқҙ м–ҙл ӨмӣҖ

    • н•ҙм„қн•ҳкі мһҗ н•ҳлҠ” м—°кө¬(XAI. Explainable AI)л“Өмқҙ мқҙм–ҙм§Җкі  мһҲмңјлӮҳ, м•„м§Ғ к°Ҳ кёёмқҙ л©ҖлӢӨ.

  • н•ҳм§Җл§Ң, л№„мҠ·н•ң нҠ№м§•мқ„ к°Җм§„ мғҳн”ҢмқҖ л№„мҠ·н•ң hidden vectorлҘј к°Җм§Ҳ кІғ

  • л§Ңм•Ҫ к°Ғ м°Ёмӣҗмқҙ лӘ…нҷ•н•ҳкІҢ н•ҳлӮҳмқҳ мқҳлҜёлҘј м§ҖлӢҢлӢӨл©ҙ к°Ғ м°Ёмӣҗмқҳ мҲ«мһҗлҘј мЎ°м Ҳн•ҳм—¬ мӣҗн•ҳлҠ” мқҙлҜём§ҖлҘј н•©м„ұн•ҙ лӮј мҲҳ мһҲмқ„ кІғ

мҡ”м•Ҫ

  • мҳӨнҶ мқёмҪ”лҚ”(AE)лҠ” м••м¶•кіј н•ҙм ңлҘј л°ҳліөн•ҳл©° нҠ№м§• м¶”м¶ңмқ„ мһҗлҸҷмңјлЎң н•ҷмҠө
    • н•„мҡ”н•ң м •ліҙмҷҖ н•„мҡ”м—ҶлҠ” м •ліҙлҘј кө¬л¶„н•  мҲҳ мһҲкІҢлҗҳлҠ” кІғ
  • мқёмҪ”лҚ”лЎңл¶Җн„° лӮҳмҳЁ мӨ‘к°„ кІ°кіјл¬ј(рқ’ӣ)мқҖ мһ…л Ҙ(рқ‘Ҙ)м—җ лҢҖн•ң feature vectorмқҙлӢӨ.
    • a.k.a Embedding vector
    • мқёмҪ”лҚ”м—җ нҶөкіјмӢңнӮӨлҠ” кІғмқҖ feature vectorм—җ лҢҖн•ң embedding кіјм •мқҙлқјкі  ліј мҲҳ мһҲмқҢ
  • Hidden layerмқҳ кІ°кіјк°’л“Өмқ„ hidden vectorsлқј л¶ҖлҘҙл©°, мқҙл“ӨмқҖ мғҳн”Ңмқҳ featureлҘј лӢҙкі  мһҲмқҢ
    • м—¬лҹ¬к°ңмқҳ hidden vectorл“Өмқ„ feature vectorлқј л¶ҖлҘј мҲҳ мһҲмқҢ
    • л”Ҙлҹ¬лӢқм—җм„ңлҠ” labelмқ„ classifyн•ҳкё° мң„н•ң feature лҘј мһҗлҸҷмңјлЎң м¶”м¶ңн•ҳм—¬ н•ҷмҠө(м „нҶөм Ғмқё лЁёмӢ лҹ¬лӢқкіјмқҳ м°Ёмқҙм җ)
  • мӢ кІҪл§қ(лҳҗлҠ” л Ҳмқҙм–ҙ)мқ„ нҶөкіјмӢңнӮӨлҠ” кІғмқҖ мһ…л Ҙ кіөк°„(input space)м—җм„ң мһ мһ¬ кіөк°„(latent space)лЎңмқҳ л§өн•‘ кіјм •
    • кі м°Ёмӣҗ кіөк°„(high-dimensional space) вҶ’ м Җм°Ёмӣҗ кіөк°„(lower-dimensional space)
  • Hidden representaionмқ„ н•ҙм„қн•ҳлҠ” кІғмқҖ л§Өмҡ° м–ҙл ӨмӣҖ
    • н•ҳм§Җл§Ң л№„мҠ·н•ң мғҳн”ҢмқҖ л№„мҠ·н•ң hidden representationмқ„ м§ҖлӢҗ кІғ!
