  1. вфевЇИ ВЋёьѓцьЁЇВ▓ўЖ░ђ Вќ┤вќц Ж▓ЃВЮИВДђ ьЎЋВЮИьЋ┤в│┤ЖИ░

    ьЂгЖ▓ї GPT Ж│ёВЌ┤ВЮў вфевЇИЖ│╝ BERT вфевЇИВЮў Ж│ёВЌ┤вАю вѓўвѕёВќ┤Вёю ВЃЮЖ░ЂьЋ┤в│┤Въљ.

    • GPT Ж│ёВЌ┤ вфевЇИ

      • GPTЖ│ёВЌ┤ВЮў LLMВЮђ ЖИ░в│ИВаЂВю╝вАю pad_tokenВЮё Ж░ђВДђЖ│а ВъѕВДђ ВЋіЖ│а, eos_tokenВЮё ВѓгВџЕьЋўВЌг ВъЁваЦ ВІюьђђВіцВЮў вЂЮВЮё вѓўьЃђвЃё

      • вћ░вЮ╝Вёю transformersвАю TokenizerвАю ьЋЎВіхВЮё ьЋўвацвЕ┤ в░ўвЊюВІю tokenizer.pad_token = tokenizer.eos_token ВЮё ВёцВаЋьЋ┤ВЋ╝ьЋювІц.

      • GPT вфевЇИВЮў ьі╣Вё▒

        • PT Ж│ёВЌ┤ВЮў вїђЖиювфе ВќИВќ┤ вфевЇИ(Large Language Models, LLM)ВЮђ ЖИ░в│ИВаЂВю╝вАю **pad_token**ВЮё ьЈгьЋеьЋўВДђ ВЋіВЮї
        • вїђВІа, ВЮ┤ вфевЇИвЊцВЮђ **eos_token**ВЮё ВѓгВџЕьЋўВЌг ВъЁваЦ ВІюьђђВіцВЮў вЂЮВЮё вѓўьЃђвЃё
        • ВЮ┤віћ ВІюьђђВіц ВЃЮВё▒ Ж│╝ВаЋВЌљВёю вфевЇИВЮ┤ ВќИВаю вгИВъЦВЮ┤ вЂЮвѓгвіћВДђвЦ╝ ВЮИВІЮьЋўвіћ вЇ░ ВцЉВџћьЋю ВЌГьЋаВЮё ьЋе

      • Tokenizer ВёцВаЋ

        • GPT Ж│ёВЌ┤ вфевЇИВЮё ВѓгВџЕьЋўВЌг ьЋЎВіхВЮё ВДёьќЅьЋа вЋї, Transformers вЮ╝ВЮ┤вИївЪгвдгВЮў **Tokenizer**вЦ╝ ВаЂВаѕьъѕ ВёцВаЋьЋ┤ВЋ╝ ьЋЕвІѕвІц. ВЮ┤вЦ╝ ВюёьЋ┤ **tokenizer.pad_token = tokenizer.eos_token**Вю╝вАю ВёцВаЋьЋ┤ВЋ╝ ьЋе(ьЋёВѕў?)

        • ВЮ┤ваЄЖ▓ї ВёцВаЋьЋеВю╝вАюВЇе, вфевЇИВЮђ ьїевћЕвљю вХђвХёВЮё ВІюьђђВіцВЮў вЂЮВю╝вАю ВЮИВІЮьЋўЖ│а, ьЋ┤вІ╣ вХђвХёВЮё вг┤ВІюьЋа Вѕў ВъѕЖ▓ї вље

        • special_tokens_map.json ВЮё в│┤вЕ┤ вІцВЮї ВёИЖ░ђВДђ в░ќВЌљ ВА┤ВъгьЋўВДђ ВЋівіћвІц.

    • BERT Ж│ёВЌ┤ вфевЇИ

      • BERT Ж│ёВЌ┤ вфевЇИВЮў ьі╣Вё▒
        • BERT Ж│ёВЌ┤ вфевЇИВЮђ ВЮ╝в░ўВаЂВю╝вАю pad_token*ВЮ┤ ВаЋВЮўвљўВќ┤ ВъѕВю╝вЕ░, ВІюьђђВіцВЮў ВІюВъЉ(*[CLS]**)Ж│╝ вЂЮ([SEP]**)ВЮё вѓўьЃђвѓ┤віћ ьі╣в│ёьЋю ьєаьЂ░ВЮё ВѓгВџЕ
      • Tokenizer ВёцВаЋ
        • BERT вфевЇИВЮў Ж▓йВџ░, **padding="max_length"**вАю ВёцВаЋьЋўвіћ Ж▓ЃВЮ┤ ВЮ╝в░ўВаЂ
        • ВЮ┤віћ вфевЊа ВІюьђђВіцЖ░ђ вЈЎВЮ╝ьЋю Вхювїђ ЖИИВЮ┤вЦ╝ Ж░ќвЈёвАЮ ьЋе
        • ВЮ┤ в░ЕВІЮВЮђ вфевЇИВЮ┤ Ж│аВаЋвљю ЖИИВЮ┤ВЮў ВъЁваЦВЮё В▓ўвдгьЋа Вѕў ВъѕЖ▓ї ьЋўвЕ░, ьі╣ьъѕ BERTВЎђ Ж░ЎВЮђ вфевЇИВЮ┤ вгИВъЦ Ж┤ђЖ│ёвЦ╝ ьЋЎВіхьЋўвіћ вЇ░ ВюаВџЕ

  2. padding в░Ј max_length ВўхВЁўВЌљ вїђьЋ┤ ВЮ┤ьЋ┤вЦ╝ ьЋ┤в│┤Въљ

    Padding ВўхВЁў

    1. True: Ж░ђВъЦ ЖИ┤ ВІюьђђВіц ЖИИВЮ┤ВЌљ вДъВХћВќ┤ вІцвЦИ вфевЊа ВІюьђђВіцвЦ╝ ьїевћЕ,ВЮ┤віћ в░░В╣ў В▓ўвдг ВІю ВюаВџЕ
    2. False вўљвіћ None: ьїевћЕВЮё ВѕўьќЅьЋўВДђ ВЋіВЮї. вфевЊа ВІюьђђВіцвіћ ВЏљвъўВЮў ЖИИВЮ┤вЦ╝ ВюаВДђ
    3. Рђюmax_lengthРђЮ: вфевЊа ВІюьђђВіцвЦ╝ max_lengthВЌљ ВДђВаЋвљю ЖИИВЮ┤вАю ьїевћЕьЋЕвІѕвІц. ВЮ┤віћ Ж│аВаЋвљю ЖИИВЮ┤ВЮў ВъЁваЦВЮё В▓ўвдгьЋўвіћ вЇ░ ВаЂьЋЕ
    4. РђюlongestРђЮ: TrueВЎђ ВюаВѓгьЋўЖ▓ї Ж░ђВъЦ ЖИ┤ ВІюьђђВіцВЌљ вДъВХћВќ┤ вѓўвеИВДђ ВІюьђђВіцвЦ╝ ьїевћЕьЋЕвІѕвІц. ьЋўВДђвДї ВЮ┤ ВўхВЁўВЮђ вфЁВІюВаЂВю╝вАю Ж░ђВъЦ ЖИ┤ ВІюьђђВіцвЦ╝ ЖИ░ВцђВю╝вАю Вѓ╝віћ Ж▓ЃВЮё Ж░ЋВА░

    Max_length ВўхВЁў

    • max_length Ж░њ ВёцВаЋ: max_lengthвіћ вфевЇИВЮ┤ В▓ўвдгьЋа Вѕў Въѕвіћ Вхювїђ ВІюьђђВіц ЖИИВЮ┤вЦ╝ ВДђВаЋ, ЖИ┤ ВъЁваЦВЮё ВаЂВаѕьЋю ьЂгЖИ░вАю ВъўвЮ╝вѓ┤віћ вЇ░ ВѓгВџЕвљўвЕ░, вЕћвфевдг ВаюьЋюВЮ┤вѓў вфевЇИ ВЋёьѓцьЁЇВ▓ўВЮў ВаюьЋюВЮё Ж│авацьЋа вЋї ВцЉВџћ
    • integer Ж░њВю╝вАю ВёцВаЋВЮё ьЋ┤вЈё вље

    ВХћЖ░ђ ВўхВЁў

    • truncation:
      • TrueвАю ВёцВаЋьЋўвЕ┤, max_lengthв│┤вІц ЖИ┤ ВІюьђђВіцвіћ ВъўвацвѓўЖ░ђВёю max_length ЖИИВЮ┤ВЌљ вДъВХ░ВДљ
      • False вўљвіћ NoneВю╝вАю ВёцВаЋьЋўвЕ┤, ВъЁваЦ ВІюьђђВіцЖ░ђ max_lengthвЦ╝ В┤ѕЖ│╝ьЋ┤вЈё ВъўвдгВДђ ВЋіВЮї
    • return_tensors:
      • ВХюваЦ вЇ░ВЮ┤ьё░ ьЃђВъЁВЮё ВДђВаЋьЋЕвІѕвІц. ВўѕвЦ╝ вЊцВќ┤, ptвіћ PyTorch ьЁљВёю, tfвіћ TensorFlow ьЁљВёю, npвіћ NumPy в░░ВЌ┤ВЮё в░ўьЎў
    • return_attention_mask:
      • TrueвАю ВёцВаЋьЋўвЕ┤, вфевЇИВЮ┤ Вќ┤вќц вХђвХёВЮ┤ ВІцВаю вЇ░ВЮ┤ьё░ВЮ┤Ж│а Вќ┤вќц вХђвХёВЮ┤ ьїевћЕВЮИВДђвЦ╝ ЖхгвХёьЋа Вѕў Въѕвіћ attention maskвЦ╝ в░ўьЎў

          (Ж░юВЮИ ВЃЮЖ░Ђ : . GPT вфевЇИВЮђ ВЌ░ВєЇВаЂВЮИ ьЁЇВіцьіИ ВЃЮВё▒ВЌљ ВхюВаЂьЎћвљўВќ┤ ВъѕВю╝в»ђвАю, ьїевћЕ ВЌєВЮ┤ ВІцВаю вЇ░ВЮ┤ьё░ВЮў ЖИИВЮ┤вЦ╝ ВюаВДђьЋўвіћ Ж▓ЃВЮ┤ вЇћ ВъљВЌ░ВіцвЪгВџ┤ Ж▓░Ж│╝вЦ╝ Ж░ђВаИВўг Вѕў ВъѕВЮё Ж▓Ѓ)

        • PaddingВЮё TrueвАю ВёцВаЋьЋўвіћ Ж▓йВџ░**:**

          2. вЈЎВаЂ ЖИИВЮ┤ВЮў ВъЁваЦ В▓ўвдг: вфевЊа ВъЁваЦВЮ┤ вЈЎВЮ╝ьЋю ЖИИВЮ┤вЦ╝ Ж░ђВДљВю╝вАюВЇе вфевЇИВЮ┤ вЇћ ВЮ╝Ж┤ђвљю в░ЕВІЮВю╝вАю вЇ░ВЮ┤ьё░вЦ╝ В▓ўвдгьЋа Вѕў ВъѕВіхвІѕвІц. ВЮ┤віћ вфевЇИВЮў ьЋЎВіхЖ│╝ ВХћвАа Вё▒віЦВЌљ ЖИЇВаЋВаЂВЮИ ВўЂьќЦВЮё Вцё Вѕў ВъѕВіхвІѕвІц.

          (Ж░юВЮИ ВЃЮЖ░Ђ : , GPTВЮў Ж▓йВџ░ ВЮ╝в░ўВаЂВю╝вАю ВЃЮВё▒ ВъЉВЌЁВЌљ ВѓгВџЕвљўЖ│а в░░В╣ў В▓ўвдгЖ░ђ ВцЉВџћьЋўВДђ ВЋіЖИ░ВЌљ FalseЖ░ђ ВюавдгьЋа Ж▓Ѓ Ж░ЎВЮї)

        • PaddingВЮё max_lengthвАю ВёцВаЋьЋўвіћ Ж▓йВџ░ Ж│аВаЋвљю ЖИИВЮ┤ВЮў ВъЁваЦ В▓ўвдг, PaddingВЮё longestвАю ВёцВаЋьЋўвіћ Ж▓йВџ░ Ж░ђВъЦ ЖИ┤ ВІюьђђВіц ЖИ░ВцђВЮў ьїевћЕВю╝вАю TrueВЎђ Ж░ЎВЮ┤ Вбђ вЇћ ВаЂВаѕьЋўВДђ ВЋіВЋё в│┤Въё

    • BERT вфевЇИВЮў Ж▓йВџ░

      • в░░В╣ў В▓ўвдгвЦ╝ ВюёьЋ┤ ВЮ╝Ж┤ђвљю ЖИИВЮ┤Ж░ђ ьЋёВџћьЋўвІцвЕ┤ max_length вўљвіћ **longest**Ж░ђ ВаЂьЋЕьЋа Вѕў ВъѕЖ│а
