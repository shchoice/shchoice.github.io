Java 8 мқҙм „м—җлҠ” м–ҙл– н•ң нҢЁлҹ¬лӢӨмһ„мқҙм—ҲлҠ”к°Җ

Java 8 л¶Җн„°лҠ” м–ҙл– н•ң нҢЁлҹ¬лӢӨмһ„м—җ мқ‘н•ҳкі  мһҲлҠ”к°Җ

Javaк°Җ кұ°л“ӯ ліҖнҷ”н•ҳлҠ” мқҙмң 

м»ҙн“ЁнҢ… нҷҳкІҪмқҳ ліҖнҷ”

  • л©ҖнӢ°мҪ”м–ҙ CPU лҢҖмӨ‘нҷ”мҷҖ к°ҷмқҖ н•ҳл“ңмӣЁм–ҙм Ғ ліҖнҷ”

Javaм—җ л¶Җм—¬лҗҳлҠ” мӢңлҢҖм Ғ ліҖнҷ” мҡ”кө¬

Java 8, Java 9мқҳ мғҲлЎңмҡҙ н•өмӢ¬ кё°лҠҘ мҶҢк°ң

н”„лЎңк·ёлһҳл°Қ м–ём–ҙ мғқнғңкі„м—җм„ң мһҗл°”мқҳ мң„м№ҳ

  • мҙҲм°Ҫкё°
    • л§ҺмқҖ мң мҡ©н•ң лқјмқҙлёҢлҹ¬лҰ¬лҘј нҸ¬н•Ён•ҳлҠ” мһҳ м„Өкі„лҗң к°қмІҙм§Җн–Ҙ м–ём–ҙлЎң м¶ңл°ң
    • мҠӨл Ҳл“ңмҷҖ лқҪмқ„ мқҙмҡ©н•ң лҸҷмӢңм„ұ м§Җмӣҗ
    • мқён„°л„· м• н”ҢлҰҝ н”„лЎңк·ёлһЁмқҳ мЈјмҡ” м–ём–ҙк°Җ лҗЁ
      • мҪ”л“ңлҘј JVM л°”мқҙнҠё мҪ”л“ңлЎң м»ҙнҢҢмқјн•ҳлҠ” нҠ№м§• л•Ңл¬ё( лӘЁл“  лёҢлқјмҡ°м Җм—җм„ң к°ҖмғҒ лЁёмӢ  мҪ”л“ңлҘј м§Җмӣҗ)
    • мәЎмҠҗнҷ” лҚ•л¶„м—җ Cм—җ л№„н•ҙ мҶҢн”„нҠёмӣЁм–ҙ м—”м§ҖлӢҲм–ҙл§Ғм Ғмқё л¬ём ңк°Җ нӣЁм”¬ м Ғм—ҲмқҢ
  • н•ҳм§Җл§Ң н”„лЎңк·ёлһҳл°Қ м–ём–ҙ мғқнғңкі„м—җ ліҖнҷ”мқҳ л°”лһҢ
    • л№…лҚ°мқҙн„°(н…Ңлқјл°”мқҙнҠё мқҙмғҒмқҳ лҚ°мқҙн„°м…Ӣ) лқјлҠ” лҸ„м „м—җ м§Ғл©ҙн•ҳл©ҙм„ң л©ҖнӢ°мҪ”м–ҙ м»ҙн“Ён„°лӮҳ м»ҙн“ЁнҢ… нҒҙлҹ¬мҠӨн„°лҘј мқҙмҡ©н•ҙм„ң л№…лҚ°мқҙн„°лҘј нҡЁкіјм ҒмңјлЎң мІҳлҰ¬н•  н•„мҡ”м„ұмқҙ м»Өм§җ
      • лі‘л ¬ н”„лЎңм„ёмӢұмқ„ нҷңмҡ©н•ҙм•ј н•ҳлҠ”лҚ°7к№Ңм§Җмқҳ мһҗл°”лЎңлҠ” м¶©л¶„нһҲ лҢҖмқ‘н•ҳкё° м–ҙл Өмӣ мқҢ
    • н•ҳм§Җл§Ң Java8мқ„ нҶөн•ҙ нҡЁкіјм ҒмңјлЎң лҢҖмқ‘
      • лі‘л ¬м„ұмқ„ нҷңмҡ©н•ҳлҠ” мҪ”л“ң, к°„кІ°н•ң мҪ”л“ңлҘј кө¬нҳ„н•  мҲҳ мһҲлҸ„лЎқ Java 8 м—җм„ң м ңкіөн•ҳлҠ” кё°лҠҘмқё 3 к°Җм§Җ н”„лЎңк·ёлһҳл°Қ к°ңл…җмқ„ м ңмӢң

Java 8 м„Өкі„мқҳ л°‘л°”нғ•мқ„ мқҙлЈЁлҠ” 3к°Җм§Җ н”„лЎңк·ёлһҳл°Қ к°ңл…җ

к°’мқ„ ліҖнҷ”мӢңнӮӨлҠ” лҚ° м§‘мӨ‘н–ҲлҚҳ кі м „м Ғмқё к°қмІҙм§Җн–Ҙм—җм„ң лІ—м–ҙлӮҳ н•ЁмҲҳнҳ• н”„лЎңк·ёлһҳл°ҚмңјлЎң лӢӨк°Җм„  кІғмқҙ Java8мқҳ к°ҖмһҘ нҒ° ліҖнҷ”

1. мҠӨнҠёлҰј мІҳлҰ¬ (Stream processing)

  • мҠӨнҠёлҰјмқҙлһҖ
    • кё°мЎҙм—җлҠ” н•ң лІҲм—җ н•ң н•ӯлӘ©мқ„ мІҳлҰ¬н–Ҳм§Җл§Ң, мҡ°лҰ¬к°Җ н•ҳл ӨлҠ” мһ‘м—…мҰқлҠҘ кі мҲҳмӨҖмңјлЎң м¶”мғҒнҷ”н•ҙм„ң мқјл Ёмқҳ мҠӨнҠёлҰјмңјлЎң л§Ңл“Өм–ҙ мІҳлҰ¬
      • нҡЁкіј : мҠӨнҠёлҰј нҢҢмқҙн”„лқјмқёмқ„ мқҙмҡ©н•ҙм„ң мһ…л Ҙ л¶Җл¶„мқ„ м—¬лҹ¬ CPU мҪ”м–ҙм—җ мүҪкІҢ н• лӢ№н•  мҲҳ мһҲмқҢ, мҰү мҠӨл Ҳл“ңлқјлҠ” ліөмһЎн•ң мһ‘м—…мқ„ мӮ¬мҡ©н•ҳм§Җ м•Ҡмңјл©ҙм„ңлҸ„ кіөм§ңлЎң лі‘л ¬м„ұмқ„ м–»мқ„ мҲҳ мһҲмқҢ
      • мҳҲм ң: cat data.csv | sort | head -n 5

2. лҸҷмһ‘ нҢҢлқјлҜён„°нҷ”(Behavior parameterization)лЎң л©”мҶҢл“ңм—җ мҪ”л“ң м „лӢ¬

Java 8м—җм„ңлҠ” л©”мҶҢл“ңлҘј лӢӨлҘё л©”мҶҢл“ңмқҳ мқёмҲҳлЎң л„ҳкІЁмЈјлҠ” кё°лҠҘмқ„ м ңкіө(кё°мЎҙм—җлҠ” л©”мҶҢл“ңлҘј лӢӨлҘё л©”мҶҢл“ңлЎң м „лӢ¬н•  л°©лІ•мқҙ м—Ҷм—ҲмқҢ)

3. лі‘л ¬м„ұкіј кіөмң  к°ҖліҖ лҚ°мқҙн„°

мҲңмҲҳ н•ЁмҲҳлҘј нҶөн•ҙ лӢӨлҘё мҪ”л“ңмҷҖ лҸҷмӢңм—җ мӢӨн–үн•ҳлҚ”лқјлҸ„ м•Ҳм „н•ҳкІҢ мӢӨн–үн•  мҲҳ мһҲмқҢ (кё°мЎҙ synchronizedлҘј мқҙмҡ©н•ҙм„ң кіөмң лҗң к°ҖліҖ лҚ°мқҙн„°лҘј ліҙнҳён•ҳлҠ” к·ңм№ҷмқ„ л§Ңл“Ө мҲҳлҠ” мһҲм§Җл§Ң мқјл°ҳм ҒмңјлЎң synchronizedлҠ” мӢңмҠӨн…ң м„ұлҠҘм—җ м•…мҳҒн–Ҙмқ„ лҜём№Ё)

вҖ» кіөмң лҗҳм§Җ м•ҠмқҖ к°ҖліҖ лҚ°мқҙн„°(no shared mutable data), л©”мҶҢл“ң, н•ЁмҲҳ мҪ”л“ңлҘј лӢӨлҘё л©”мҶҢл“ңлЎң м „лӢ¬н•ҳлҠ” кё°лҠҘмқҖ н•ЁмҲҳнҳ• нҢЁлҹ¬лӢӨмһ„мқҳ н•өмӢ¬м Ғмқё мӮ¬н•ӯ (л°ҳл©ҙ лӘ…л №нҳ• н”„лЎңк·ёлһҳл°Қ(imperative programming) нҢЁлҹ¬лӢӨмһ„мқҖ мқјл Ёмқҳ к°ҖліҖ мғҒнғңлЎң н”„лЎңк·ёлһЁмқ„ м •мқҳ), мҰү мқҙ мҡ”кө¬мӮ¬н•ӯмқҖ мҲҳн•ҷм—җм„ңмқҳ н•ЁмҲҳмІҳлҹј н•ЁмҲҳк°Җ м •н•ҙм§„ кё°лҠҘл§Ң мҲҳн–үн•ҳм—¬ лӢӨлҘё л¶Җмһ‘мҡ©мқҖ мқјмңјнӮӨм§Җ м•ҠмқҢмқ„ мқҳлҜё

мһҗл°”к°Җ м§„нҷ”н•ҙм•ј н•ҳлҠ” мқҙмң 

ліҖнҷ”лҗң н•ҳл“ңмӣЁм–ҙлӮҳ н”„лЎңк·ёлһҳлЁёмқҳ кё°лҢҖм—җ л¶Җмқ‘н•ҳкё° мң„н•Ё, кё°мЎҙ к°’мқ„ ліҖнҷ”мӢңнӮӨлҠ” лҚ° м§‘мӨ‘н–ҲлҚҳ кі м „м Ғмқё к°қмІҙм§Җн–Ҙм—җм„ң лІ—м–ҙлӮҳ н•ЁмҲҳнҳ• н”„лЎңк·ёлһҳл°ҚмңјлЎң лӢӨк°Җм„°лҚҳ кІғмқҙ мһҗл°”8мқҳ к°ҖмһҘ нҒ° ліҖнҷ”
