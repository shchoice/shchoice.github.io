Profile 수정

프로필 아바타 이미지 및 소개 수정

  • cart.styl

    • 경로 : themes/icarus/include/style

    • 아바타 이미지에 요소검사를 수행하면 figure 태그를 수정하면 됨을 알 수 있음, 따라서 card-content > image 를 수정

    • 제목 및 소개 글의 요소 검사를 수행하면 p태그를 수정함을 알 수 있음, card-content > level-item > p 의 1,2,3 번째 요소를 수정

Article 수정

Title 폰트 사이즈/굵기 및 본문과 간격 변경

  • article.styl

    • 경로 : themes/icarus/include/style

    • 제목에 요소 검사를 수행하면 article tag 하위에 css class로 title이 걸려있음을 확인할 수 있음, 따라서 아래 내용처럼 수정

본문의 h1, h2 굵기 변경

  • article.styl

    • 경로 : themes/icarus/include/style

    • article tag 하위에 content class 밑에 있는 h1, h2 태그를 수정

styl 파일 수정 후, hexo server로 확인 시 css 파일 지우기

  • styl 파일을 수정 후 hexo generate만 수행해서 사용한다면 pulbic/css/*.css 파일을 지워서 사용해야 함(파일이 있을 경우 수정 내용 반영 안됨)
  • 위를 방지하고 싶으면 hexo clean && hexo generate을 하자

Google Analytics 기능 추가

  • Google Analytics 에 Tracking ID 찾기

    • (애널리틱스 계정 생성 및 속성 생성 생략)

    • 관리 > 데이터 스트림

    • tracking ID 복사

    [상세내용] Googel Analytics Tracking ID 설정

  • config_icarus.yml 파일에서 google_analytics > tracking_id 란에 복사한 Tracking ID 붙여넣기

post글에 댓글 기능 추가(by disqus)

  • Disqus 회원 가입Disqus 및 Site 생성(생략)

  • Short Name 복사

  • config_icarus.yml 파일에서 comment > shorname에 복사 내용 붙여넣기
