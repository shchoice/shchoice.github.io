    • ьЋўВДђвДї ВЌгвЪг ВЊ░ваѕвЊюЖ░ђ Ж│хВюављю вЕћвфевдг Ж│хЖ░ёВЮё ВѓгВџЕьЋўЖИ░ВЌљ вЇ░ВЮ┤ьё░ вХѕВЮ╝В╣ў вгИВаювЦ╝ Ж│авацьЋ┤ВЋ╝ьЋе

    • вфевЊа Java ьћёвАюЖиИвъеВЮђ ЖИ░в│ИВаЂВю╝вАю ьЋўвѓўВЮў ВЊ░ваѕвЊю(вЕћВЮИ ВЊ░ваѕвЊю)вЦ╝ Ж░ќВЮї, main methodвЦ╝ ВІцьќЅьЋўЖ│а, ьћёвАюЖиИвъеВЮ┤ ВбЁвБївља вЋїЖ╣їВДђ ВІцьќЅвље

  2. ВъљВЏљ Ж│хВюа : CPU, Memory, ьїїВЮ╝, вёцьіИВЏїьЂг вЊ▒ВЮў ВІюВіцьЁю ВъљВЏљВЮё Ж│хВюа

  3. ВіцВ╝ђВцёвДЂ : вфевЉљ ВіцВ╝ђВцёвДЂВЮў вїђВЃЂВЮ┤ вље, ВдЅ Вџ┤ВўЂВ▓┤ВаюЖ░ђ ВъљВЏљВЮё ьЋавІ╣ьЋўвіћ вїђВЃЂВЮ┤ вље
  4. вЈЎВІюВё▒ : ьћёвАюВёИВіцВЎђ ВЊ░ваѕвЊювіћ вфевЉљ вЈЎВІюВЌљ ВІцьќЅвља Вѕў ВъѕВю╝вЕ░, ВЮ┤віћ вЕђьІ░ьЃюВіцьѓ╣ВЮё ЖхгьўёьЋўЖИ░ ВюёьЋю ьЋёВѕўВаЂВЮИ ЖИ░віЦ

  • ВЮ╝в░ўВаЂВю╝вАювіћ RunnableВЮё ЖхгьўёьЋю Ж░ЮВ▓┤вЦ╝ Thread ВЃЮВё▒ВъљВЮў ВЮИВъљвАю ВаёвІгьЋўвіћ в░ЕВІЮВю╝вАю ВЃѕвАюВџ┤ ВЊ░ваѕвЊювЦ╝ ВЃЮВё▒

  1. Race Condition(Ж▓йВЪЂ ВЃЂьЃю)

      • ВъљвБїВЮў ВЮ╝Ж┤ђВё▒ВЮё ьЋ┤В╣ўвіћ Ж▓░Ж│╝Ж░ђ вѓўьЃђвѓа Вѕў ВъѕЖИ░ вЋївгИ
    • ВЮ┤вЦ╝ ьЋ┤Ж▓░ьЋўЖИ░ ВюёьЋ┤Вёювіћ вЈЎЖИ░ьЎћ ЖИ░в▓ЋВЮё ВѓгВџЕ

    • вїђВІа ВЊ░ваѕвЊювЦ╝ ВбЁвБїВІюьѓцЖИ░ ВюёьЋ┤Вёювіћ ьћївъўЖиИ в│ђВѕўвѓў interrupt() вЕћВєївЊювЦ╝ ВѓгВџЕьЋўвіћ Ж▓ЃВЮ┤ ВбІВЮї
  4. Thread Вџ░ВёаВѕюВюё ВДђВаЋ ВБ╝ВЮў

    • ВЊ░ваѕвЊюв│ёвАю вЈЁвдйВаЂВЮИ вЇ░ВЮ┤ьё░вЦ╝ ВађВъЦьЋўвіћвЇ░ ВѓгВџЕвљўвіћ ThreadLocalВЮё вѓев░юьЋўвЕ┤ ВЊ░ваѕвЊю Ж░ёВЌљ вЇ░ВЮ┤ьё░ вХѕВЮ╝В╣ў вгИВаюЖ░ђ в░юВЃЮьЋа Вѕў ВъѕВЮї
