Neural Language Model

  • Find parameter that maximize likelihood for given training corpus

    • $D={x^i }_{i=1}^N$ .

      $\hat{\theta} = \text{argmax}{\theta \in \Theta} \sum{i=1}^{N} \log P(x_{1:n}^{i}; \theta)$ // лҚ°мқҙн„°м—җ лҢҖн•ҙ log-likelihoodлҘј мөңлҢҖнҷ”н•ҳлҠ” нҢҢлқјлҜён„°лҘј м°ҫмһҗ

      $= \text{argmax}{\theta \in \Theta} \sum{i=1}^{N}\sum_{j=1}^{n} \log P(x_{j}^{i}|x_{<j}^i; \theta)$
      // by chain-rule
      $\text{where } x_{1:n} = {x_1, вҖҰ, x_n}$

  • Take a step of gradient descent to minimize negative log-likelihood

    • $L(\theta) = -\sum_{i=1}^{N}\sum_{j=1}^{n} \log P(x_{j}^{i},|P(x_{<j}^i; \theta)$

      $\theta \leftarrow \theta - \eta\nabla_\theta L(\theta)$
      $\log P(x_t | x_{<t}; \theta) = x_t^T \cdot \log f_\theta(x_{t-1}, h_{t-1})$ // softmax layerлҘј нҶөкіјн•ҳлҠ” кІҪмҡ°,
      // $x_t^T$ : GT (One-hot vector, мӢӨм ң мҳҲмёЎ лӢЁм–ҙ)
      // $\log f_\theta(x_{t-1}, h_{t-1})$ : LM(softmax layerм—җ log м”Ңмҡҙ нҷ•лҘ к°’)

    $\text{where } x_t \text{ is one-hot vector, and }f_Оё \text{ is model with parameter Оё.}$

    • лҚ” м„ёл¶Җм ҒмңјлЎң ліҙл©ҙ.

      • $f(x_{t-1}, h_{t-1}) = \text{softmax}(RNN(\text{emb}(x_{t-1}), h_{t-1}) \cdot W)$ $\text{, where W} \in \mathbb{R}^{hidden-size \times |V|}$ // WлҠ” softmax м „м—җ мғқлһөлҗң linear layer
        $=\text{softmax}(h_t \cdot W)\text{, where } W \in \mathbb{R}^{hidden-size \times |V|}$

        $=\hat{x_t}$

        $\text{where } \hat{x_t} \text{ is a probability distribution that } P(\cdot|x_{<t};\theta)$

        • $\hat{рқ‘Ҙ_рқ‘Ў}$ : mini-batch лӮҙ л¬ёмһҘлі„ лӢЁм–ҙлі„ нҷ•лҘ к°’, мқҙм „лӢЁм–ҙк°Җ мЈјм–ҙмЎҢмқ„ л•Ң м„ёнғҖм—җм„ңмқҳ нҢҢлқјлҜён„°лҘј к°–мқҖ нҷ•лҘ  л¶„нҸ¬

        • |w| = (hs, |V|)

        • |$h_t$|=(рқ‘Ҹрқ‘ , в„Һрқ‘ ) = (bs,1, hs)

        • |$\hat{x_t}$| = (bs,1,hs) x (hs, |V|) = (bs, |V|)

  • softmaxлҠ” лӢӨмқҢ time-stepм—җ лӮҳмҳ¬ лӢЁм–ҙм—җ лҢҖн•ң нҷ•лҘ  л¶„нҸ¬лҘј лӮҳнғҖлғ„ мҰү, softmax layerмқҳ к°Ғ elementлҠ” лӢӨмқҢ лӢЁм–ҙмқҳ нҷ•лҘ к°’(л¶„нҸ¬)(discrete multinomial distribution)

Evaluation

  • Perplexity (PPL) лһҖ

    • н…ҢмҠӨнҠё л¬ёмһҘм—җ лҢҖн•ҙм„ң м–ём–ҙлӘЁлҚёмқ„ мқҙмҡ©н•ҳм—¬ нҷ•лҘ (likelihood)мқ„ кө¬н•ҳкі 

    • PPL мҲҳмӢқм—җ л„Јм–ҙ м–ём–ҙлӘЁлҚёмқҳ м„ұлҠҘ мёЎм •

      • л¬ёмһҘмқҳ нҷ•лҘ мқ„ кёёмқҙм—җ лҢҖн•ҙм„ң normalization (кё°н•ҳнҸүк· )

        • $PPL(x_1, \ldots, x_n; \theta) = P(x_1, \ldots, x_n; \theta)^{-\frac{1}{n}} = \sqrt[n]{ \frac{1}{P(x_1, \ldots, x_n; \theta)}}$

          • лӢЁм–ҙл“Өмқҳ Chain Rule мқҙкё°м—җ л¬ёмһҘмқҙ кёёмҲҳлЎқ нҷ•лҘ кіұмңјлЎң к°’мқҙ мһ‘м•„м§җ(1ліҙлӢӨ мһ‘мқҖ к°’мқҙкё° л•Ңл¬ё)
          • м§§мқҖ л¬ёмһҘм—җ л№„н•ҙ кёҙ л¬ёмһҘмқҖ нҷ•лҘ мқҙ мһ‘м•„м§Җкі  кёёмқҙм—җ л”°лҘё кё°н•ҳнҸүк· мқ„ н•ҳкё°м—җ(-1/n) кёёмқҙм—җ мғҒкҙҖм—Ҷмқҙ normalizeн•  мҲҳ мһҲмқҢ
          • нҷ•лҘ м—җ м—ӯмҲҳлҘј м·Ён–ҲмңјлҜҖлЎң PPLмқҖ мһ‘мқ„мҲҳлЎқ м„ұлҠҘмқҙ мўӢмқҢмқ„ мқҳлҜё

  • Chain Ruleм—җ мқҳн•ҙм„ң

    • $PPL(x_1, \ldots, x_n; \theta) = P(x_1, \ldots, x_n; \theta)^{-\frac{1}{n}} =\sqrt[n]{\frac{1}{P(x_1, \ldots, x_n; \theta)}} =\sqrt[n]{\frac{1}{\prod_{i=1}^{n} P(x_i | x_{<i}; \theta)}}$

  • Markov assumptionмқҙ м Ғмҡ© лҗ  кІҪмҡ°

    • $PPL(x_1, \ldots, x_n; \theta) = P(x_1, \ldots, x_n; \theta)^{-\frac{1}{n}} = \sqrt[n]{ \frac{1}{P(x_1, \ldots, x_n; \theta)}} =\sqrt[n]{\frac{1}{\prod_{i=1}^{n} P(x_i | x_{<i}; \theta)}} \approx \sqrt[n]{\frac{1}{\prod_{i=1}^{n} P(x_i | x_{i-1}, \ldots, x_{i-k}; \theta)}}$

  • Perplexity

    • н…ҢмҠӨнҠё л¬ёмһҘм—җ лҢҖн•ҙм„ң нҷ•лҘ мқ„ лҶ’кІҢ л°ҳнҷҳн• мҲҳлЎқ мўӢмқҖ м–ём–ҙлӘЁлҚё

    • н…ҢмҠӨнҠё л¬ёмһҘм—җ лҢҖн•ң PPLмқҙ мһ‘мқ„мҲҳлЎқ мўӢмқҖ м–ём–ҙлӘЁлҚё

    • мҳҲм ң

      • мЈјмӮ¬мң„лҘј лҚҳм ёлҙ…мӢңлӢӨ.

        • 1л¶Җн„° 6к№Ңм§Җмқҳ 6к°ңмқҳ мҲ«мһҗлЎң мқҙлЈЁм–ҙм§„ мҲҳм—ҙ
        • 1л¶Җн„° 6к№Ңм§Җ 6к°ңмқҳ мҲ«мһҗмқҳ м¶ңнҳ„ нҷ•лҘ мқҖ лӘЁл‘җ к°ҷмқҢ

      • uniform distribution

        • $D={x^i}_{i=1}^{N} \text{, where } x_i \sim P(x) \text{ and } \forall x \in {1, 2, 3, 4, 5, 6}$

          $PPL(x_1, \ldots, x_n; \theta) =\sqrt[n]{\frac{1}{P(x_1, \ldots, x_n; \theta)}}=\sqrt[n]{\frac{1}{\prod_{i=1}^{n} P(x_i)}}$ // лҸ…лҰҪмӢңн–үмқҙкё°м—җ
          $=\sqrt[n]{\frac{1}{(\frac{1}{6})^n}} = 6$ // мЈјмӮ¬мң„ л©ҙмқҳ к°ҜмҲҳ

    • PerplexityлҘј н•ҙм„қн•ҳлҠ” л°©лІ•

      • мЈјмӮ¬мң„ PPL : л§Ө time-step к°ҖлҠҘн•ң к°Җм§“мҲҳмқё 6
      • л»—м–ҙлӮҳк°Ҳ мҲҳ мһҲлҠ” branch(к°Җм§Җ)мқҳ мҲ«мһҗлҘј мқҳлҜё
      • Time-step лі„ нҸүк·  branchмқҳ мҲҳ
      • PPLмқҙ лӮ®мқ„мҲҳлЎқ нҷ•лҘ  л¶„нҸ¬к°Җ Sharp н•ҳлӢӨ.
      • PPLмқҙ лҶ’мқ„мҲҳлЎқ нҷ•лҘ  л¶„нҸ¬к°Җ Flat н•ҳлӢӨ.

Perplexity & Entropy

  • Cross Entropy

    • $H(P, P_\theta) = -E_{x_{1:n} \sim P}[\log P(x_{1:n}; \theta)]$ // P(X)м—җм„ң м–ҙл–Ө л¬ёмһҘмқ„ samplingн•ҳм—¬ мҡ°лҰ¬ лӘЁлҚём—җ л„Јм—Ҳмқ„ л•ҢлЎңк·ё нҷ•лҘ к°’мқ„ нҸүк· лӮҙкі  л§Ҳмқҙл„ҲмҠӨлҘј м·Ён•ҳлҠ” CE

      $\approx -\frac{1}{n} \sum_{x_{1:n} \in X} P(x_{1:n}) \log P(x_{1:n}; \theta)\text{, defined as per-word entropy}$ //P(x_{1:n})\log{P(x_{1:n}; \theta)}: GT x Log-likelihood

    $\approx -\frac{1}{n \times N} \sum_{i=1}^{N} \log P(x_{1:n}^{i}; \theta) \text{, by Monte-Carlo}\approx -\frac{1}{n} \log P(x_{1:n}; \theta)\text{, where N=1}$ // 1к°ңмқҳ л¬ёмһҘмңјлЎң CEлҘј кө¬н•Ё
    $\approx -\frac{1}{n} \sum_{i=1}^{N} \log P(x_i | x_{<i}; \theta)$ // by chain rule
    $=L(x_{1:n}; \theta)$

    • $L(x_{1:n}; \theta) \approx -\frac{1}{n} \sum_{i=1}^{N} \log P(x_i | x_{<i}; \theta)=-\frac{1}{n} \log \prod_{i=1}^{n} P(x_i | x_{<i}; \theta)=\log\sqrt[n]{\frac{1}{\prod_{i=1}^{n} P(x_i | x_{<i}; \theta)}}=\log{PPL(x_{1:n};\theta)}$

  • мҰү PPL exp(CE)мҷҖ к°ҷлӢӨ.

    • equivalent to minimize cross entropy

  • GT л¶„нҸ¬мҷҖ LM л¶„нҸ¬ мӮ¬мқҙмқҳ cross entropyлҘј кө¬н•ҳкі  minimize
    • л¬ёмһҘмқҳ perplexityлҘј minimize.
  • perplexityлҘј нҶөн•ҙ н—·к°ҲлҰ¬лҠ” лӢЁм–ҙл“Өмқҳ мҲ«мһҗмқҙкё°м—җ, лӮҙ LMмқҙ л§Ө time-stepл§ҲлӢӨ н—·к°ҲлҰ¬лҠ” лӢЁм–ҙл“Өмқҳ мҲ«мһҗк°Җ лӘҮ к°ңкө¬лӮҳлқјлҠ” кІғмқ„ м•Ң мҲҳ мһҲмқҢ.

    • P(,I, Love, to, play, ) $\approx \frac{COUNT(,I, Love, to, play, )}{COUNT(,I, Love, to, play)}$

Markov Assumptionмқ„ м Ғмҡ©н•ҳл©ҙ

  • Apporximate with counting only previous k tokens (лӘЁл“  лӢЁм–ҙлҘј ліҙм§Җ м•Ҡкі  kк°ңл§Ң лҙ„)

    • $P(x_n|x_{<n}) \approx P(x_n|x_1,вӢҜ,x_{n-k}) \approx \frac{COUNT(x_{n-k},вӢҜ,x_{n})}{COUNT(x_{n-k},вӢҜ,x_{n-1})}$

  • if k = 2

    • $P(x_n|x_{<n}) \approx P(x_n|x_{n-2},x_{n-1},x_{n}) \approx \frac{COUNT(x_{n-2},x_{n-1},x_{n})}{COUNT(x_{n-2},x_{n-1})}$

Smoothing & Discounting

  • Smoothing
    • Markov assumptionмқ„ лҸ„мһ…н•ҳмҳҖм§Җл§Ң м—¬м „нһҲ л¬ём ңлҠ” лӮЁм•„мһҲмқҢ
    • Training corpusм—җ м—ҶлҠ” unseen word sequenceмқҳ нҷ•лҘ мқҖ 0?
    • Unseen word sequenceм—җ лҢҖн•ң лҢҖмІҳ
      • Smoothing or Discounting
    • Popular algorithm
      • Modified Kneser-Ney Discounting

  • Add One Smoothing

    • To prevent count becomes zero

      • $P(w_t|w_{<t}) \approx \frac{C(w_{1:t})}{C(w_{1:t-1})}$ // t time-stepмқҳ мӢңм җмқј л•Ңк№Ңм§Җмқҳ t time-stepмқҳ м–ём–ҙмқҳ нҷ•лҘ  л¶„нҸ¬(нҷ•лҘ к°’) $\approx \frac{C(w_{1:t}) + 1}{C(w_{1:t-1}) + ,|V,|} text{where } |V| \text{ is a size of vocabulary}$

    • Generalization of Add One Smoothing

      • If we generalize this :

        • $P(w_t|w_{<t}) \approx \frac{C(w_{1:t})}{C(w_{1:t-1})} \approx \frac{C(w_{1:t}) + 1}{C(w_{1:t-1}) + ,|V,|} \approx \frac{C(w_{1:t}) + k}{C(w_{1:t-1}) + k \times ,|V,|} \approx \frac{C(w_{1:t}) + \frac{m}{,|V,|}}{C(w_{1:t-1}) + m} \text{where } |V| \text{ is a size of vocabulary}$

      • Take more generailzation :

        • $P(w_t|w_{<t}) \approx \frac{C(w_{1:t}) + m \times P(w_t)}{C(w_{1:t-1}) + m}$

          $\text{where } P(w_t) \text{ is unigram probability}$

  • лӢӨм–‘н•ң лӢЁм–ҙ л’Өм—җ лӮҳнғҖлӮҳлҠ” лӢЁм–ҙмқјмҲҳлЎқ unseen word sequenceм—җ л“ұмһҘн•  нҷ•лҘ мқҙ лҶ’м§Җ м•Ҡмқ„к№Ң?

    • м•һм—җ л“ұмһҘн•ң лӢЁм–ҙмқҳ мў…лҘҳк°Җ лӢӨм–‘н• мҲҳлЎқ н•ҙлӢ№ нҷ•лҘ мқҙ лҶ’мқ„ кІғ к°ҷмқҢ

      $P_{continuation} (w) \propto | {v : C(v, w) > 0} |$

      • $P_{continuation}$ : лӢЁм–ҙ $w$ мқҙнӣ„м—җ кі„мҶҚлҗҳлҠ” лӢЁм–ҙмқҳ нҷ•лҘ мқ„ лӮҳнғҖлғ„
      • $C(v, w)$ : лӢЁм–ҙ $v$мҷҖ $w$ к°Җ н•Ёк»ҳ л“ұмһҘн•ҳлҠ” нҡҹмҲҳлҘј лӮҳнғҖлғ„
      • $\propto$ : л№„лЎҖ
      • $| {v : C(v, w) > 0} |$лҠ” $w$мҷҖ н•Ёк»ҳ л“ұмһҘн•ҳлҠ” лӢЁм–ҙ $v$мқҳ мҲҳлҘј лӮҳнғҖлғ„

Interpolation (ліҙк°„лІ•)

  • лӢӨлҘё Language Modelмқ„ linearн•ҳкІҢ мқјм • л№„мңЁ(рққҖ)лЎң м„һлҠ” кІғ
  • general domain LM + domain specific LM = general domainм—җм„ң мһҳ лҸҷмһ‘н•ҳлҠ” domain adapted LM
  • domain specific LMExamples
    • мқҳлЈҢ domain ASR, MT system
    • лІ•лҘ  domain ASR, MT system
    • нҠ№н—Ҳ domain MT system
  • мҲҳмӢқ
    • $\tilde{P}(w_n | w_{n-k}, \ldots, w_{n-1}) = \lambda P_1(w_n | w_{n-k}, \ldots, w_{n-1}) + (1 - \lambda) P_2(w_n | w_{n-k}, \ldots, w_{n-1})$
      • $P_1(w_n | w_{n-k}, \ldots, w_{n-1})$ : мІ«лІҲм§ё LM
      • $P_2(w_n | w_{n-k}, \ldots, w_{n-1})$ : л‘җлІҲм§ё LM
  • к·ёлғҘ domain specific corpusлЎң LMмқ„ л§Ңл“Өл©ҙ мһҘл•Ў м•„лӢҢк°Җ?
    • к·ёлҹј unseen word sequenceк°Җ л„Ҳл¬ҙ л§Һмқ„ мҲҳ мһҲмқҢ
  • к·ёлғҘ м „мІҙ corpusлҘј н•©міҗм„ң LMмқ„ л§Ңл“Өл©ҙ мһҘл•Ў м•„лӢҢк°Җ?
    • Domain Specific corpusмқҳ м–‘мқҙ л„Ҳл¬ҙ м Ғм–ҙм„ң л°ҳмҳҒмқҙ м•Ҳлҗ  мҲҳ мһҲмқҢ
  • Interpolationм—җм„ң ratio(рққҖ)лҘј мЎ°м Ҳн•ҳм—¬ мӨ‘мҡ”лҸ„(weight)лҘј мЎ°м Ҳ
    • лӘ…мӢңм Ғ(explicit)мңјлЎң м„һмқ„ мҲҳ мһҲлӢӨ.
    • General domain test setкіј Domain specific test set лӘЁл‘җм—җм„ң мўӢмқҖ м„ұлҠҘмқ„ м°ҫлҠ” hyper-parameter рқңҶлҘј м°ҫм•„м•ј н•ңлӢӨ.
  • мҳҲм ң
    • вҖңмӨҖл№„ лҗң м§„м •м ң лҘј нҲ¬м—¬ н•© мӢңлӢӨвҖң
    • General domain

  • We can convert joint probability yo conditional probability

    • $P(A,B,C,D)=P(D|A,B,C)P(A,B,C)

  • By Chain Rule,
