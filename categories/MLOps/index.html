MLflow к°ңмҡ” л°Ҹ м„Өм№ҳлІ•

л°°кІҪ

мғҲлЎң л“Өм–ҙмҳӨлҠ” DL н”„лЎңм қнҠём—җ лҢҖн•ҙ л§ӨлІҲ мӢӨн—ҳмқ„ мҲҳкё°(PPT, мҠӨн”„л Ҳл“ңмӢңнҠё)лЎң мҲҳн–үн•ҳлҠ” лҚ°м—җ н•ңкі„лҘј лҠҗлӮҢ

  • ліөмҲҳмқҳ лӘЁлҚё л°Ҹ м—¬лҹ¬ н•ҳмқҙнҚј нҢҢлқјлҜён„° мқҳ к°’мқ„ мЎ°м •н•ҙ к°Җл©° н…ҢмҠӨнҠёлҘј мҲҳн–үн•ҙ мөңм Ғнҷ”лҗң лӘЁлҚёмқ„ мӮ°м¶ңн•ҳкі  мӢ¶м—ҲмқҢ

    • Epoch, Layer Depth, Regularization(Dropout, Batch Norm, max_len_size, num_update_token л“ұ)

  • м •нҷ•лҸ„ нҸүк°Җ м§Җн‘ңлҘј м—¬лҹ¬к°ң мӮ¬мҡ©н•ҳкі  к·ё кІ°кіјлҘј н•ңлҲҲм—җ ліҙкё°лҘј нқ¬л§қ

    • л¶„лҘҳ : Accuracy, Recall, Precision, F1-Score, AUROC л“ұ
    • мҳҲмёЎ : MAE, RMSE, MPE, R2 Score л“ұ

  • мӮ°м¶ңн•ң лӘЁлҚёмқҳ лІ„м „ кҙҖлҰ¬к°Җ м–ҙл ӨмӣҖ

    • ex) model_loss .pkl л“ұ

м„Өм№ҳ нҷҳкІҪ

  • OS : Ubuntu 16.04
  • Python : 3.8
    • (python 3.6 мқҙмғҒмқҙл©ҙ л¬ём ң м—ҶмқҢ)

MLflow к°ңмҡ”

  • MLflowлһҖ

    • MLflowлҠ” мӢӨн—ҳ(experimentation), мһ¬нҳ„м„ұ(reproducibility), л°°нҸ¬(deployment) л°Ҹ Model registry л“ұмқҳ ML life-cycleмқ„ кҙҖлҰ¬н•  мҲҳ мһҲлҠ” мҳӨн”ҲмҶҢмҠӨ н”Ңлһ«нҸј
    • CLI л°Ҹ мӣ№ кё°л°ҳ GUI лҘј м§Җмӣҗ
    • Tensorflow, PyTorch, Keras, sklearn, fast.ai л“ұ лӢӨм–‘н•ң лқјмқҙлёҢлҹ¬лҰ¬мҷҖ нҶөн•©н•ҳм—¬ мӮ¬мҡ©н•  мҲҳ мһҲмқҢ
    • л¶Җл¶„ мң лЈҢмқё WanDB л“ұ лӢӨлҘё нҲҙлҸ„ мЎҙмһ¬н•ҳм§Җл§Ң, л¬ҙлЈҢлЎң мӮ¬мҡ©н•  мҲҳ мһҲмқҢ

  • мЈјмҡ”кё°лҠҘ(нҒ¬кІҢ 4к°Җм§Җ ComponentлЎң кө¬м„ұ)

    1. MLflow Tracking
      • ML CodeлҘј мҲҳн–үн•  л•Ң м„Өм •н•ң parameter, code version, metris, loss мқҳ ліҖнҷ”, мӢңк°Ғнҷ” мһҗлЈҢ л“ұмқҳ logлҘј API л°Ҹ UIлЎң м ңкіө
      • Tensorflow & Keras, pytorch, FastAI, Scikit-learn, Spark, XGBoost л“ұмқҳ лқјмқҙлёҢлҹ¬лҰ¬м—җ лҢҖн•ҙм„ңлҠ” мһҗлҸҷмңјлЎң лЎңк·ёлҘј кё°лЎқн•Ё(Automatic Logging)
    2. MLflow Projects
      • ML codeлҘј мһ¬мӮ¬мҡ©, мһ¬кө¬нҳ„ к°ҖлҠҘн•ң нҳ•нғңлЎң нҢЁнӮӨм§•н•ҳм—¬ лӢӨлҘё лҚ°мқҙн„° кіјн•ҷмһҗл“Өкіј кіөмң н•ҳкұ°лӮҳ н”„лЎңлҚ•м…ҳмңјлЎң ліҖнҷҳ кё°лҠҘ м ңкіө
    3. MLflow Models
      • REST API кё°л°ҳмңјлЎң мӢӨмӢңк°„ м„ңл№ҷ л°Ҹ м¶”лЎ  н”Ңлһ«нҸј м ңкіө
    4. Model Registry
      • Centeralized model storeмқҳ м—ӯн• мқ„ мҲҳн–үн•Ё
      • Modelмқҳ мғқм„ұл¶Җн„° лӘЁлҚёмқҳ лІ„м „ л°Ҹ мҠӨн…Ңмқҙм§ҖлҘј м¶©лҸҢм—Ҷмқҙ кҙҖлҰ¬н•  мҲҳ мһҲмқҢ
        л”°лқјм„ң Model Registryм—җ updateк°Җ л°ңмғқн•  кІҪмҡ°, н•өмӢ¬ м •ліҙлҘј лЎңк·ёлЎң лӮЁкёё мҲҳ мһҲлҸ„лЎқ мқҙлІӨнҠёлҘј м„Өкі„

  • REST API л°Ҹ CLI лҘј нҶөн•ҙ лӘЁл“  кё°лҠҘм—җ м—‘м„ёмҠӨн•  мҲҳ мһҲмқҢ

  • APIлҠ” Pyhthon, R, Java л“ұ мЎҙмһ¬

MLflow м„Өм№ҳ л°Ҹ кё°лҸҷ

  • MLflow м„Өм№ҳ л°Ҹ нҷ•мқё

  • MLflow л””л үнҶ лҰ¬ мғқм„ұ

    • к°„лӢЁн•ң кө¬м„ұмқ„ мң„н•ҙ кі„м • home path м—җ mlflow л””л үнҶ лҰ¬лҘј кө¬м„ұн•ҳкі  н•ҙлӢ№ л””л үнҶ лҰ¬м—җ mlflowмқҳ artifact л°Ҹ data нҢҢмқјмқ„ м ҖмһҘ

  • MLflow tracking server кё°лҸҷ

    • mlflow ui лЎң кё°лҸҷн•ҳлҠ” л°©лІ•, mlflow server лЎң кё°лҸҷн•ҳлҠ” л°©лІ• мҙқ 2к°Җм§Җ л°©лІ•мқҙ мһҲмқҢ

      • mlflow uiлҠ” localhostм—җм„ңл§Ң кё°лҸҷ к°ҖлҠҘ

        • мһ‘м—… м—°мҶҚм„ұмқ„ мң„н•ҙм„ңлҠ” кё°лҸҷмқҖ л°ҳл“ңмӢң к°ҷмқҖ л””л үнҶ лҰ¬м—җм„ң мҲҳн–үн•ҙм•јн•Ё

      • mlflow serverлҠ” production мҡ©лҸ„лЎң мӮ¬мҡ©

        • workerлҘј м—¬лҹ¬к°ң лқ„мҡё мҲҳ мһҲкі  prometheusк°Җ metricsлҘј к°Җм ёк°Ҳ мҲҳ мһҲлҠ” м—”л“ңнҸ¬мқёнҠёлҘј м ңкіөн•ҳлҠ” м¶”к°Җм Ғ кё°лҠҘ мЎҙмһ¬

      • мҳөм…ҳ м„ӨлӘ…

        • -h : м ‘мҶҚмқҙ к°ҖлҠҘн•ң л„ӨнҠёмӣҢнҒ¬ мЈјмҶҢ

        • -p : м ‘мҶҚм—җ мӮ¬мҡ©н•  нҸ¬нҠёлІҲнҳё

        • вҖ“backend-store-uri

          • MLflow м—”нӢ°нӢ°к°Җ кё°лЎқлҗҳлҠ” м ҖмһҘмҶҢ (experiments мқҙлҰ„, params, metrics л“ұ)
          • DBлӮҳ нҢҢмқјмӢңмҠӨн…ңмңјлЎң м ҖмһҘмқҙ лҗҳл©° м•„л¬ҙ м„Өм • м—Ҷмқ„мӢң ./mlrunsм—җ лҚ°мқҙн„°к°Җ лЎңк№…мқҙ лҗңлӢӨ.

            • DB : sqlite:///path/to/file.db

            • нҢҢмқјмӢңмҠӨн…ң : file:///absolute/path/to/directory

        • вҖ“default-artifact-root

          • ML experimentм—җ лҢҖн•ң artifactsк°Җ м ҖмһҘмқҙ лҗЁ

            • backend-store-uriлҘј dbлЎң м„Өм •н• мӢң artifactл§Ң лӢӨмқҢ нҢҢмқјмӢңмҠӨн…ңм—җ м ҖмһҘлҗЁ

            • backend-store-uriлҘј filesystem мңјлЎң м„Өм •н•  мӢң artifact к°Җ entityмҷҖ н•Ёк»ҳ м ҖмһҘлҗЁ

          • нҢҢмқјмӢңмҠӨн…ңмңјлЎң м ҖмһҘмқҙ лҗҳл©° м•„л¬ҙм„Өм • м—Ҷмқ„мӢң ./mlrunsм—җ лҚ°мқҙн„°к°Җ лЎңк№…мқҙ лҗңлӢӨ.

      • м¶”к°Җ мҳөм…ҳмқ„ мӮ¬мҡ©н•ҳкі  мӢ¶мқ„мӢң mlflow server вҖ“help лЎң мғҒм„ё нҷ•мқё

  • MLflow tracking server UI нҷ•мқё

    • http://<ip>:<port> лЎң м ‘мҶҚ

лҠҗлӮҖм җ

MLflowлҘј м„Өм№ҳ мӢң, backend-store-uri лҠ” sqlite, mariadb л“ұмқҳ RDBMS м—җ л“ұлЎқмқ„ н•ҙм•ј м¶”нӣ„ кҙҖлҰ¬н•ҳкё°м—җ мҡ©мқҙн•ң кІғ к°ҷлӢӨ.
