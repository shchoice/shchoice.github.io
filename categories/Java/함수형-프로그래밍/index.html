NPE(NullPointerException, н•Ём •м№ҙл“ң к°ҷмқҖ м—җлҹ¬)

Optional (мһҲмқ„ мҲҳлҸ„ мһҲкі  м—Ҷмқ„ мҲҳлҸ„ мһҲлӢӨ.)

  • Empty : л№Ҳ Optionalмқ„ л§Ңл“Ө л•Ң

  • ofNullable : Nullмқём§Җ м•„лӢҢм§Җ м•Ңм§Җ лӘ»н•ҳлҠ” мҳӨлёҢм қнҠёлЎң Optionalмқ„ л§Ңл“Ө л•Ң

    • к°’мқҙ мһҲлҠ” кІҪмҡ°

    • к°’мқҙ м—ҶлҠ” кІҪмҡ°

Optional м•Ҳм—җ мһҲлҠ” к°’мқ„ нҷ•мқён•ҳкі  кәјлӮҙлҠ” лІ•

  • isPresent - м•Ҳмқҳ мҳӨлёҢм қнҠёк°Җ nullмқём§Җ м•„лӢҢм§Җ мІҙнҒ¬, Nullмқҙ м•„лӢҗ мӢң true
  • get - Optional м•Ҳмқҳ м•–мқ„ м¶”м¶ң, Null мқҙлқјл©ҙ м—җлҹ¬
  • ofElse - Optionalмқҙ nullмқҙ м•„лӢҲлқјл©ҙ Optional м•Ҳмқҳ к°’мқ„, nullмқҙлқјл©ҙ otherлЎң кіөкёүлҗң к°’мқ„ лҰ¬н„ҙ
  • orElseGet вҖ“ Optionalмқҙ nullмқҙ м•„лӢҲлқјл©ҙ Optional м•Ҳмқҳ к°’мқ„, nullмқҙлқјл©ҙ supplierлЎң кіөкёүлҗҳлҠ” к°’мқ„ лҰ¬н„ҙ
  • orElseThrow вҖ“ Optionalмқҙ nullмқҙ м•„лӢҲлқјл©ҙ Optional м•Ҳмқҳ к°’мқ„, nullмқҙлқјл©ҙ exceptionSupplierлЎң кіөкёүлҗҳлҠ” exceptionмқ„ лҚҳм§җ

Optional мқ‘мҡ©мқ„ мң„н•ҙ м•Ңм•„м•јн•  кІғл“Ө

  • map вҖ“ Optionalмқҙ nullмқҙ м•„лӢҲлқјл©ҙ mapperлҘј м Ғмҡ©, nullмқҙлқјл©ҙ лӢЁмҲңнһҲ л№„м–ҙмһҲлҠ” Optional мҰү м•„лһҳ мҳҲм ңм—җм„ңлҠ” falseлҘј л°ҳнҷҳ

  • flatMap вҖ“ mapperмқҳ лҰ¬н„ҙ к°’мқҙ лҳҗ лӢӨлҘё Optionalмқҙлқјл©ҙ н•ң лӢЁкі„мқҳ Optionalмқҙ лҗҳлҸ„лЎқ лӮ©мһ‘н•ҳкІҢ н•ҙмӨҢ

  3. мһ¬мӮ¬мҡ©м„ұ л°Ҹ мң м§ҖліҙмҲҳм„ұ мҰқк°Җ: н•ЁмҲҳлҘј 1кёү мӢңлҜјмңјлЎң м·Ёкёүн•ҳл©ҙ, нҠ№м • кё°лҠҘмқ„ мҲҳн–үн•ҳлҠ” мҪ”л“ң мЎ°к°Ғмқ„ н•ЁмҲҳлЎң л¶„лҰ¬н•ҳкі  мқҙлҘј м—¬лҹ¬ кіім—җм„ң мһ¬мӮ¬мҡ©н•  мҲҳ мһҲкІҢ лҗЁ. мқҙлҠ” мҪ”л“ң мӨ‘ліөмқ„ мӨ„мқҙкі  мң м§ҖліҙмҲҳлҘј мҡ©мқҙн•ҳкІҢ л§Ңл“¬
  4. м„ м–ём Ғ н”„лЎңк·ёлһҳл°Қ: н•ЁмҲҳлҘј лҚ°мқҙн„°мҷҖ л§Ҳм°¬к°Җм§ҖлЎң м·Ёкёүн•ҳл©ҙ, н”„лЎңк·ёлһЁмқ„ ліҙлӢӨ м„ м–ём ҒмңјлЎң н‘ңнҳ„н•  мҲҳ мһҲ. мқҙлҠ” н”„лЎңк·ёлһЁмқҳ нқҗлҰ„мқ„ лҚ” лӘ…нҷ•н•ҳкІҢ н•ҳкі , к°ңл°ңмһҗк°Җ мқҳлҸ„н•ҳлҠ” л°”лҘј м§ҒкҙҖм ҒмңјлЎң м „лӢ¬н•  мҲҳ мһҲкІҢ н•ҙ
