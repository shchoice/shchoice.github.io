2мһҘ. Probabilistic Perspective - MAP

MAP(Maximum A Posterior) Estimation

  • Dataк°Җ мЈјм–ҙмЎҢмқ„ л•Ң к°Җм„Ө hлҘј maximize н•ҙм•јн•ңлӢӨ. мқҙлҘј мң„н•ҙ Posteriorмқ„ maximizeн•  мҲҳлҸ„ мһҲкі  likelihoodлЎң maximizeн•  мҲҳлҸ„ мһҲмқҢ

  • likelihoodлҘј maximize н•ңлӢӨл©ҙ MLE

  • мқҙ мӢңк°„м—җлҠ” poeteriror лҘј maximizeн•ҙм„ң hлҘј м°ҫлҸ„лЎқ н•ҳкІ мқҢ вҶ’ MAP

  • MAPлҘј мқҙн•ҙн•ҳкё° мң„н•ң мҳҲм ң

    • м ҲлҸ„ мӮ¬кұҙмқҳ лІ”мқёмқҖ л°ңмһҗкөӯмқ„ лӮЁкІјмҠөлӢҲлӢӨ.

    • мӢ л°ң мӮ¬мқҙмҰҲ 240мқј л•Ң, лІ”мқёмқҖ лӮЁмһҗмқјк№Ң? м—¬мһҗмқјк№Ң?

      • $P(\text{y}в”Ӯ\text{x} =240)$

    • м§Җмқё мӨ‘м—җ мӢ л°ң мӮ¬мқҙмҰҲк°Җ 240мқҙм—ҲлҚҳ мӮ¬лһҢл“Өмқ„ л– мҳ¬л Өліҙмһҗ

      • м—¬мһҗ мӨ‘м—җ л§Һмқ„к№Ң? лӮЁмһҗ мӨ‘м—җ л§Һмқ„к№Ң?
        • м—¬мһҗ мӨ‘м—җм„ң мӢ л°ң мӮ¬мқҙмҰҲк°Җ 240мқё мӮ¬лһҢ мқҙлқјкі  мғқк°Ғн•ҳл©ҙ likelihood(к°Җм •мқҙ л“Өм–ҙк°„ кІғ
        • $P(\text{x}=240|\text{y})$

    • к·ёлҹ°лҚ° лІ”н–ү мһҘмҶҢк°Җ кө°л¶ҖлҢҖлқјл©ҙ?

      • $P(\text{y}=\text{male}) > P(\text{y}=\text{female})$

        // 99% > 1%

    • LikelihoodлҠ” м—¬мһҗмқј к°ҖлҠҘм„ұмқҙ лҶ’м§Җл§Ң, PriorлҘј кі л Өн•ҳмҳҖмқ„ л•Ң лІ”мқёмқҖ лӮЁмһҗмқј к°ҖлҠҘм„ұмқҙ лҶ’мқҢ

      • $P(\text{y} | \text{x} = 240) = \frac{P(\text{x} = 240 | \text{y})P(\text{y})}{P(\text{x} = 240)}$
      • $\frac{P(\text{x} = 240 | \text{y=male})P(\text{y=male})}{P(\text{x} = 240)} > \frac{P(\text{x} = 240 | \text{y=female})P(\text{y=female})}{P(\text{x} = 240)}$

  • MAP Estimation

    • Find $\hat{h}$, which maximizes posterior

      • $\hat{h} = \text{argmax}_{h \in H} P(h|D)$

        $= \text{argmax}_{h \in H} \frac{P(D|h)P(h)}{P(D)}$ // л°‘ліҖмқҖ hмҷҖ мғҒкҙҖ м—Ҷм–ҙ лӮ л ӨлІ„лҰј

        $= \text{argmax}_{h \in H}P(D|h)P(h)$ // рқ‘ғ(рқҗ·|в„Һ) : likelihood, рқ‘ғ(в„Һ) : prior

  • MAP Estimation (л”Ҙлҹ¬лӢқмқҙлӢҲ hлҘј рқңҪлЎң л°”кҝҲ)

    • Bayesian кҙҖм җ

      $\hat{\theta} = \text{argmax}{\theta \in \Theta} P(\theta|D) = \text{argmax}{\theta \in \Theta} \frac{P(D|\theta)P(\theta)}{P(D)} = \text{argmax}_{\theta \in \Theta}P(D|\theta)P(\theta)$ // Dataк°Җ мЈјм–ҙмЎҢмқ„ л•Ң рқңғк°Җ к¶ҒкёҲн•Ё

      • нҢҢлқјлҜён„° лҳҗн•ң Random VariableлЎң к°„мЈјн•ҳл©°,
      • мӮ¬м „ м •ліҙ(prior)лҘј нҶөн•ҙ нҢҢлқјлҜён„°мқҳ мӮ¬м „ л¶„нҸ¬лҘј м„Өм • (priorм—җ лҢҖн•ң к°Җм •мқҙ н•„мҡ”)
      • мқҙнӣ„ кҙҖм°°лҗң лҚ°мқҙн„°лҘј нҶөн•ҙ мӮ¬нӣ„(posterior) л¶„нҸ¬лҘј кі„мӮ°н•ҳл©°, мӮ¬нӣ„ л¶„нҸ¬к°Җ мөңлҢҖк°Җ лҗҳлҠ” нҢҢлқјлҜён„°лҘј м°ҫлҠ” MAP л°©лІ•мқ„ мӮ¬мҡ©
      • лҜёлһҳмқҳ uncertainty к№Ңм§Җ кі л Ө
      • likelihoodлҘј maximizeн•ҳлҠ” лҸҷмӢңм—җ priorк№Ңм§Җ maximize н•Ё

    • Frequentist кҙҖм җ

      $\hat{\theta} = \text{argmax}_{\theta \in \Theta}P(D;\theta)$ //# рқ‘ғ(рқңғ,рқҗ·), likelihoodлҘј maximize

      • нҢҢлқјлҜён„°лҠ” кі м •лҗң к°’мқҙл©° мөңм Ғнҷ”мқҳ лҢҖмғҒмңјлЎң лҙ„
      • кҙҖм°°лҗң лҚ°мқҙн„°лҘј кё°л°ҳмңјлЎң MLE л°©лІ•мқ„ мӮ¬мҡ©н•ҳм—¬ нҢҢлқјлҜён„°лҘј м¶”м •
      • кіјм Ғн•©(Overfitting)м—җ м·Ём•Ҫн•Ё

Summary

  • MAPлҘј нҶөн•ҙ мҡ°лҰ¬лҠ” posteriorлҘј мөңлҢҖнҷ” н•ҳлҠ” hypothesisлҘј м°ҫмқ„ мҲҳ мһҲмқҢ
    • л§Ҳм°¬к°Җм§ҖлЎң мЈјм–ҙм§„ лҚ°мқҙн„°м…Ӣм—җ лҢҖн•ң posteriorлҘј мөңлҢҖнҷ” н•ҳлҠ” нҢҢлқјлҜён„°(рқңғ)лҘј м°ҫмқ„ мҲҳ мһҲмқҢ
  • Bayesian кҙҖм җм—җм„ңлҠ” нҢҢлқјлҜён„°л“Өмқ„ лһңлҚӨ ліҖмҲҳлЎң ліҙкі  priorм—җ лҢҖн•ң к°Җм •мқ„ нҶөн•ҙ, м•һмңјлЎңмқҳ uncertainty к№Ңм§Җ кі л Ө
    • мқҙлҘј нҶөн•ҙ overfitting л“ұмқҳ л¬ём ңлҸ„ н•ҙкІ°н•  мҲҳ мһҲмқҢ
  • Bayesian Deep Learningм—җ лҢҖн•ң лӢӨм–‘н•ң мӢңлҸ„л“ӨлҸ„ мқҙм–ҙм§Җкі  мһҲмқҢ
    • лІ мқҙм§Җм•Ҳ л°©лІ•лЎ мқҖ кі„мӮ°лҹүмқҙ л§Һмқҙ н•„мҡ”н•ҳкі  ліөмһЎн•ң л”Ҙлҹ¬лӢқ лӘЁлҚём—җ м Ғмҡ©н•ҳлҠ” лҚ°м—җлҠ” м—¬м „нһҲ м—¬лҹ¬ лҸ„м „м Ғмқё л¬ём ңл“Өмқҙ мЎҙмһ¬

2мһҘ. Probabilistic Perspective - мӢ кІҪл§қкіј MLE

Again, our object is

  • лҚ°мқҙн„°лҘј л„Јм—Ҳмқ„ л•Ң м¶ңл Ҙмқ„ л°ҳнҷҳн•ҳлҠ” к°ҖмғҒмқҳ н•ЁмҲҳлҘј лӘЁмӮ¬н•ҳлҠ” кІғ
  • нҷ•лҘ  л¶„нҸ¬лЎңл¶Җн„° мғҳн”Ңл§Ғн•ҳм—¬ лҚ°мқҙн„°лҘј л„Јм—Ҳмқ„ л•Ң, нҷ•лҘ  л¶„нҸ¬лҘј л°ҳнҷҳн•ҳлҠ” к°ҖмғҒмқҳ н•ЁмҲҳлҘј лӘЁмӮ¬н•ҳлҠ” кІғ
    • м¶ңл Ҙ л¶„нҸ¬м—җм„ң мғҳн”Ңл§Ғн•ҳл©ҙ мӣҗн•ҳлҠ” м¶ңл Ҙ к°’мқ„ м–»мқ„ мҲҳ мһҲлӢӨ.
  • Example: мҶҗ кёҖм”Ёк°Җ мЈјм–ҙмЎҢмқ„ л•Ң, кёҖм”Ёмқҳ нҒҙлһҳмҠӨмқҳ нҷ•лҘ  л¶„нҸ¬
    • $P(c|x) \text{, where } x \sim P(x)$

Before we start,

  • лӘЁл‘җ к°ҷмқҖ н‘ңнҳ„ (рқңғ лҘј рқ‘…рқ‘Һрқ‘‘рқ‘ңрқ‘ҡ рқ‘үрқ‘Һрқ‘ҹрқ‘–рқ‘Һрқ‘Ҹрқ‘ҷрқ‘’лЎң м·Ёкёү)
    • $P_{\theta}(x) = P(x; \theta) = P(x|\theta)$ рқңғ лқјлҠ” нҢҢлқјлҜён„°лҘј к°–лҠ” нҷ•лҘ л¶„нҸ¬м—җм„ңмқҳ рқ‘Ҙмқҳ нҷ•лҘ к°’
    • $P_{\theta}(y|x) = P(y|x; \theta) = P(y|x, \theta)$ рқңғ лқјлҠ” нҢҢлқјлҜён„°лҘј к°–лҠ” нҷ•лҘ л¶„нҸ¬м—җм„ңмқҳ рқ‘Ҙк°Җ мЈјм–ҙмЎҢмқ„ л•Ң рқ‘Ұмқҳ нҷ•лҘ к°’(

Parameters for Probability Distribution

  • Bernoulli Distribution
    • рқңғ={рқ‘қ} (лҸҷм „лҚҳм§Җкё°мқҳ нҷ•лҘ  к°ҷмқҖ)
  • Gaussian Distribution
    • рқңғ={рқңҮ,рқңҺ} (лҢҖн•ңлҜјкөӯ нӮӨ(мӢ мһҘ)мқҳ л¶„нҸ¬лҘј м•Ңкі  мӢ¶м–ҙмҡ”)
  • л”Ҙлҹ¬лӢқм—җм„ңлҠ” нҢҢлқјлҜён„°к°Җ WмҷҖ b к°Җ лҗҳлҠ” кІғмқҙлӢӨ!! мқҙ WмҷҖ bлҘј нҶөн•ҙ нҷ•лҘ  л¶„нҸ¬ н•ЁмҲҳлҘј м •мқҳн•  мҲҳ мһҲмқҢ

Parameters for Deep Neural Networks

  • $\theta = {W_1, b_1, W_2, b_2, \ldots, W_l, b_l}$
  • л§Ҳм°¬к°Җм§ҖлЎң Gradient AscentлҘј нҶөн•ҙ LikelihoodлҘј мөңлҢҖлЎң н•ҳлҠ” нҢҢлқјлҜён„°(рқңҪ)лҘј м°ҫмқ„ мҲҳ мһҲлӢӨ.
    • $\sum_{i=1}^{n} \log P(x_i | \theta)$ лҘј Maximize н•ҳлҠ” рқңғлҘј м°ҫмқ„ мҲҳ мһҲмқҢ

Negative Log Likelihood(NLL)

  • н•ҳм§Җл§Ң лҢҖл¶Җл¶„мқҳ л”Ҙлҹ¬лӢқ н”„л Ҳмһ„мӣҢнҒ¬л“ӨмқҖ Gradient Descentл§Ң м§Җмӣҗ
    • $\theta \leftarrow \theta - \alpha \cdot \frac{\partial \mathcal{L}(\theta)}{\partial \theta}$
  • л”°лқјм„ң maximization л¬ём ңм—җм„ң minimization л¬ём ңлЎң м ‘к·ј (-1л§Ң кіұн•ҳл©ҙ лҗЁ)

Deep Neural Networks with MLE

  • MLEлҠ” кҙҖмёЎлҗң лҚ°мқҙн„°лҘј к°ҖмһҘ мһҳ м„ӨлӘ…н•ҳлҠ” лӘЁлҚё нҢҢлқјлҜён„°лҘј м°ҫлҠ” л°©лІ•мңјлЎң, лӘЁлҚёмқҳ мҶҗмӢӨ н•ЁмҲҳлҘј м„Өкі„н•ҳлҠ” лҚ° мӮ¬мҡ©лҗҳл©°, мқјл°ҳмІҷмңјлЎң cross-entropy loss function л°Ҹ NLLмқҙ MLEлҘј кё°л°ҳмңјлЎң н•Ё. мқҙ мҶҗмӢӨ н•ЁмҲҳлҘј мөңмҶҢнҷ” н•ЁмңјлЎңмҚЁ лӘЁлҚёмқҖ лҚ°мқҙн„°лҘј к°ҖмһҘ мһҳ м„ӨлӘ…н•ҳлҠ” нҢҢлқјлҜён„° к°’мқ„ м°ҫм•„к°ҖлҠ” кіјм •мқ„ мҲҳн–ү
  • MLEлҠ” нҠ№м • лҚ°мқҙн„°лҘј к°ҖмһҘ мһҳ м„ӨлӘ…н•ҳлҠ” нҷ•лҘ  лӘЁлҚёмқҳ л§Өк°ңліҖмҲҳлҘј м¶”м •н•ҳлҠ” л°©лІ•
  • кҙҖм°°лҗң лҚ°мқҙн„°м—җ лҢҖн•ң лӘЁлҚёмқҳ м Ғн•©м„ұмқ„ мёЎм •
  • л¶„нҸ¬ $P(\text{x})$лЎңл¶Җн„° мғҳн”Ңл§Ғн•ң лҚ°мқҙн„° рқ’ҷк°Җ мЈјм–ҙмЎҢмқ„ л•Ң, нҢҢлқјлҜён„° рқңғлҘј к°–лҠ” DNNмқҖ мЎ°кұҙл¶Җ нҷ•лҘ  л¶„нҸ¬лҘј лӮҳнғҖлғ„
    • $P(\text{y}|x;\theta) \text{, where } x \sim P(\text{x})$
  • мқҙл•Ң, мҡ°лҰ¬лҠ” Gradient DescentлҘј нҶөн•ҙ NLLмқ„ мөңмҶҢнҷ”н•ҳлҠ” рқңҪлҘј м°ҫмқ„ мҲҳ мһҲмқҢ
    • $\hat{\theta} = \text{argmin}{\theta \in \Theta} \sum{i=1}^{N} -\log P(y_i | x_i; \theta)$

Summary

  • MLEлҘј нҶөн•ҙ мҲҳм§‘н•ң лҚ°мқҙн„°м…Ӣмқ„ мһҳ м„ӨлӘ…н•ҳлҠ” нҷ•лҘ  л¶„нҸ¬мқҳ нҢҢлқјлҜён„°лҘј м°ҫмқ„ мҲҳ мһҲмқҢ
    • н•ҳм§Җл§Ң л№„м„ нҳ• н•ЁмҲҳм—җм„ңлҠ” лІ лҘҙлҲ„мқҙлӮҳ к°Җмҡ°мӢңм•Ҳ нҷ•лҘ л¶„нҸ¬лЎңлҠ” м„ӨлӘ…н•  мҲҳ м—ҶмқҢ
    • л¶„лҘҳ мһ‘м—…мқҳ кІҪмҡ°м—җлҠ” л„ӨнҠёмӣҢнҒ¬мқҳ м¶ңл Ҙмқ„ softmax н•ЁмҲҳм—җ нҶөкіјмӢңмјң к°Ғ нҒҙлһҳмҠӨм—җ мҶҚн•  нҷ•лҘ мқ„ лӮҳнғҖлӮҙлҠ” нҷ•лҘ  л¶„нҸ¬лҘј м–»мқ„ мҲҳ мһҲмқҢ
    • мқҙм§„ л¶„лҘҳ лӘЁлҚём—җм„ңлҠ” мӢңк·ёлӘЁмқҙл“ң н•ЁмҲҳлҘј мӮ¬мҡ©н•  мҲҳ мһҲмқҢ(м¶ңл Ҙ лІ”мң„лҠ” 0~1)
  • Neural Networks лҳҗн•ң нҷ•лҘ  л¶„нҸ¬ н•ЁмҲҳмқҙлҜҖлЎң, MLEлҘј нҶөн•ҙ нҢҢлқјлҜён„°(рққҒ, рққҲ)лҘј м°ҫмқ„ кІғ
    • мөңлҢҖн•ң лҢҖмӢ  мөңмҶҢнҷ”лҘј мң„н•ҙ Negative Log-Likelihood(NLL)мқ„ Gradient Descent
  • Gradient DescentлҘј мҲҳн–үн•ҳкё° мң„н•ҙм„ , нҢҢлқјлҜён„°м—җ лҢҖн•ң лҜёл¶„мқҙ н•„мҡ”н•Ё
    • мқҙлҘј нҡЁмңЁм ҒмңјлЎң мҲҳн–үн•ҳкё° мң„н•ҙ back-propagationмқ„ нҷңмҡ©

л”Ҙлҹ¬лӢқмқҙлһҖ нҷ•лҘ  л¶„нҸ¬ н•ЁмҲҳмқё лҸҷмӢңм—җ лӮ®мқҖ м°ЁмӣҗмңјлЎң м°Ёмӣҗмқ„ м¶•мҶҢн•ҳмһҗ

MLE Equation

  • $D={(x_i, y_i)}_{i=1}^{N} \text{, where } x_i: \sim P(\text{x}) \text{, } y_i= \sim P(\text{y} ,|x_i)$
  • $\hat{\theta} = \text{argmax}{\theta \in \Theta} \sum{i=1}^{N} \log P(y_i | x_i; \theta)$$= \text{argmin}{\theta \in \Theta} -\sum{i=1}^{N} \log P(y_i | x_i; \theta)$
  • $\theta \leftarrow \theta - \eta \nabla_\theta L(\theta)$

Connection to Deep Neural Networks

  • $D={(x_i, y_i)}_{i=1}^{N}$
  • $\hat{\theta} = \text{argmin}{\theta \in \Theta} -\sum{i=1}^{N} \log P(y_i | x_i; \theta)$
  • вҶ’ $\hat{y_i}= f_\theta(x_i)$
  • $-\sum_{i=1}^{N} \log P(y_i | x_i; \theta)=-\sum_{i=1}^{N} y_i^T \cdot \log\hat{y_i}$

MLE(NLL) & Cross Entropy Loss

  • Minimizing Negative Log-Likelihood is equal to Minimizing Cross Entropy

  • $CE(y_{1:N},\hat{y}{1:N}) = -\frac{1}{N} \sum{i=1}^{N} y_i^T \cdot \log \hat{y}_i$ // (1/рқ‘ҒмқҖ лӮҳмӨ‘м—җ рқңғлЎң лҜёл¶„лҗҳм–ҙм„ң мӮ¬лқјм§Җкё°м—җ кІ°көӯ MLEмҷҖ к°ҷмқҢ)

    $= -\frac{1}{N} \cdot \sum_{i=1}^{N} \sum_{j=1}^{d} y_{i,j} \times \log \hat{y}_{i,j}$

    $= -\frac{1}{N} \cdot \sum_{i=1}^{N} \log P_\theta (y_i|x_i)$

    $\text{where } y_{1:N} \in \mathbb{R}^{N \times d} \text{ and } \hat{y}_{1:N} \in \mathbb{R}^{N \times d}$

  • л¶„лҘҳл¬ём ңм—җм„ң Cross EntropyлҘј мӮ¬мҡ©н•ҳкі  CE LossлҘј мӮ¬мҡ©н•ҳм—¬ мөңм Ғнҷ”лҘј мҲҳн–үн–ҲлҠ”лҚ°, мқҙ лӘЁл“  кІғмқҖ кІ°көӯ MLEлҘј мҲҳн–үн•ҳкё° мң„н•ң кІғмқҙлӢӨ.(рқңҪлҘј мһҳ м°ҫлҠ” кіјм •(лӮҙк°Җ мҲҳм§‘н•ң лҚ°мқҙн„°лҘј к°ҖмһҘ мһҳ лӮҳнғҖлӮҙкё° мң„н•ң кіјм •)мқ„ н•ҳлҠ” кІғ)

  • Cross-Entropy мҶҗмӢӨ н•ЁмҲҳлҠ” л‘җ нҷ•лҘ  л¶„нҸ¬ мӮ¬мқҙмқҳ м°ЁмқҙлҘј мёЎм •

    • л”Ҙлҹ¬лӢқм—җм„ңлҠ” мқјл°ҳм ҒмңјлЎң н•ҳлӮҳмқҳ л¶„нҸ¬лҠ” мӢӨм ң л Ҳмқҙлё”мқ„ лӮҳнғҖлӮҙкі , лӢӨлҘё н•ҳлӮҳлҠ” лӘЁлҚёмқҳ мҳҲмёЎмқ„ лӮҳнғҖлғ„
    • мқҙ кІҪмҡ° Cross-Entropy мҶҗмӢӨ н•ЁмҲҳлҠ” лӘЁлҚёмқҳ мҳҲмёЎмқҙ мӢӨм ң л Ҳмқҙлё”мқ„ м–јл§ҲлӮҳ мһҳ л°ҳмҳҒн•ҳлҠ”м§ҖлҘј мёЎм •н•ҳлҠ” лҚ° мӮ¬мҡ©

Negative Log-Likelihood (NLL)мҷҖ Cross-Entropy мҶҗмӢӨ н•ЁмҲҳлҠ” л№„мҠ·н•ң к°ңл…җмқ„ нҸ¬н•Ён•ҳкі  мһҲм§Җл§Ң, м—„л°ҖнһҲ л§җн•ҙм„ң лҸҷмқјн•ң кІғмқҖ м•„лӢҳ

мқҙ л‘ҳмқҳ м°Ёмқҙм җмқҖ NLLмқҙ лӢЁмқј нҷ•лҘ  л¶„нҸ¬м—җ лҢҖн•ң мёЎм •мқ„ м ңкіөн•ҳлҠ” л°ҳл©ҙ, Cross-EntropyлҠ” л‘җ нҷ•лҘ  л¶„нҸ¬ мӮ¬мқҙмқҳ м°ЁмқҙлҘј мёЎм •н•ңлӢӨлҠ” м җмһ…лӢҲлӢӨ. мқҙ м°Ёмқҙ л•Ңл¬ём—җ, мқҙ л‘җ мҶҗмӢӨ н•ЁмҲҳлҠ” лӢӨлҘё л¬ёл§Ҙм—җм„ң мӮ¬мҡ©лҗҳл©°, к·ё кІ°кіјлҸ„ м•Ҫк°„ лӢӨлҘј мҲҳ мһҲмқҢ

к·ёлҹ¬лӮҳ л¶„лҘҳ л¬ём ңм—җм„ңлҠ” л‘җ мҶҗмӢӨ н•ЁмҲҳк°Җ лҸҷмқјн•ң кІ°кіјлҘј м ңкіөн•ҳлҠ” кІҪмҡ°к°Җ л§ҺмҠөлӢҲлӢӨ. мқҙлҠ” к°Ғ нҒҙлһҳмҠӨм—җ лҢҖн•ң мӢӨм ң л Ҳмқҙлё” л¶„нҸ¬к°Җ мӣҗ-н•« мқёмҪ”л”© нҳ•мӢқмқҙкё° л•Ңл¬ё л”°лқјм„ң NLLкіј Cross-Entropy мҶҗмӢӨ н•ЁмҲҳлҠ” мў…мў… мғҒнҳё көҗнҷҳм ҒмңјлЎң мӮ¬мҡ©

2мһҘ. Probabilistic Perspective - Maximum Likelihood Estimation

Maximum Likelihood Estimation(MLE)

Maximum Likelihood Estimation лһҖ

  • лӢӨмқҢмқҳ мҳҲмӢңлҘј нҶөн•ҙ мҷң MLEк°Җ н•„мҡ”н•ңм§Җ м•Ңм•„ліҙмһҗ

    • лӢӨмқҢкіј к°ҷмқҙ лҢҖн•ңлҜјкөӯ нӮӨмқҳ нҸүк· мқ„ м–»кё° мң„н•ҙ м•„лһҳмҷҖ к°ҷмқҖ мғҳн”Ңмқ„ м–»м—ҲлӢӨкі  к°Җм •н•ҙліҙмһҗ. мқҙмқҳ нҸүк· кіј н‘ңмӨҖнҺём°ЁлҘј к°ҖмһҘ мһҳ м–»кІҢ н•ҳл Өл©ҙ м–ҙл–»кІҢ н•ҙм•јн• к№Ң?

    • лӢӨмқҢкіј к°ҷмқҙ 3к°ңмқҳ к°Җмҡ°мӢңм•Ҳ л¶„нҸ¬(Gaussian Distribution)лҘј мӮҙнҺҙліҙмһҗ. м–ҙл–Ө к°Җмҡ°мӢңм•Ҳ л¶„нҸ¬к°Җ к°ҖмһҘ мҡ°лҰ¬мқҳ мғҳн”Ңмқ„ мһҳ м„ӨлӘ…н–ҲлӢӨкі  н•  мҲҳ мһҲмқ„к№Ң?

      • м җм„ мқҳ кёёмқҙ(LineвҖҷs length = probabilty density)к°Җ к°ҖмһҘ кёҙ кІғмқҙ м ңмқј мһҳ н‘ңнҳ„н•ң кІғ к°ҷмқҖлҚ°!(Approxmation 3)

    • л”°лқјм„ң мҡ°лҰ¬мқҳ мғҳн”Ңмқ„ к°ҖмһҘ мһҳ н‘ңнҳ„н•ҳлҠ” к°Җмҡ°мӢңм•Ҳ л¶„нҸ¬лҘј к·ёлҰ¬кё° мң„н•ң рқңҮмҷҖ рқңҺлҘј м°ҫмһҗ

      • Likelihood лҘј мөңлҢҖнҷ” н•ҳлҠ” MLE(Maximum Likelihood Estimation)мқ„ кө¬н•ҳмһҗ

  • Likelihood Function

    • мһ…л ҘмңјлЎң мЈјм–ҙм§„ нҷ•лҘ  л¶„нҸ¬(нҢҢлқјлҜён„°(рқңҮ,рқңҺ))к°Җ лҚ°мқҙн„°лҘј м–јл§ҲлӮҳ мһҳ м„ӨлӘ…н•ҳлҠ”м§Җ лӮҳнғҖлӮҙлҠ” м җмҲҳ(Likelihood)лҘј м¶ңл Ҙ н•ҳлҠ” н•ЁмҲҳ
      • мһ…л Ҙ : нҷ•лҘ  л¶„нҸ¬лҘј н‘ңнҳ„н•ҳлҠ” нҢҢлқјлҜён„°(рқңҮ,рқңҺ)
      • м¶ңл Ҙ : лҚ°мқҙн„°лҘј м„ӨлӘ…н•ҳлҠ” м •лҸ„ (Probability Densityк°Җ лҶ’м•„м•ј)
    • лҚ°мқҙн„°лҘј мһҳ м„ӨлӘ…н•ҳлҠ”м§Җ м•Ң мҲҳ мһҲлҠ” л°©лІ•
      • лҚ°мқҙн„°к°Җ н•ҙлӢ№ нҷ•лҘ  л¶„нҸ¬м—җм„ң лҶ’мқҖ нҷ•лҘ  к°’мқ„ к°Җм§Ҳ кІғ
    • мҲҳмӢқ
      • $L(\theta | x) = P(x | \theta)$
        • $x$ : кҙҖм°°лҗң лҚ°мқҙн„°
        • $\theta$ : лӘЁлҚёмқҳ нҢҢлқјлҜён„°
        • $P(x|\theta) : \theta$лЎң мЈјм–ҙм§„ $x$мқҳ нҷ•лҘ 
        • $L : \theta$ м—җ лҢҖн•ң likjelihood function(мҡ°лҸ„ н•ЁмҲҳ)
      • $L(\theta | x_1, x_2, вҖҰ, x_n) = \prod_{i=1}^{n} P(x_i | \theta)$ (мЈјм–ҙм§„ лҚ°мқҙн„°м…Ӣмқҙ лҸ…лҰҪм Ғмқҙкі  лҸҷмқјн•ҳкІҢ л¶„нҸ¬(i.i.d)н•ң кІҪмҡ°, м „мІҙ лҚ°мқҙн„°мқҳ likelhood functionмқҖ к°Ғк°Ғмқҳ лҚ°мқҙн„° нҸ¬мқёнҠём—җ лҢҖн•ң нҷ•лҘ мқҳ кіұмңјлЎң н‘ңнҳ„лҗ  мҲҳ мһҲмқҢ)
        • н•ҳм§Җл§Ң мӢӨм ң кі„мӮ°м—җм„ңлҠ” мқҙл ҮкІҢ кіұм…Ҳ нҳ•нғңмқҳ мҡ°лҸ„ н•ЁмҲҳлҠ” кі„мӮ°мқҙ м–ҙл өкі , underflow л¬ём ңлҘј мқјмңјнӮӨкё° л•Ңл¬ём—җ, лҢҖл¶Җл¶„мқҳ кІҪмҡ° лЎңк·ё мҡ°лҸ„ н•ЁмҲҳ(log-likelihood function)мқ„ мӮ¬мҡ© (мӨ‘мҡ”)
          • лЎңк·ёлҘј м·Ён•ҳл©ҙ кіұм…Ҳмқҙ лҚ§м…ҲмңјлЎң л°”лҖҢм–ҙ кі„мӮ°мқҙ мү¬мӣҢм§җ $\log L(\theta | x_1, x_2, вҖҰ, x_n) = \sum_{i=1}^{n} \log P(x_i | \theta)$

  • лӢӨмқҢмқҳ мҳҲм ңлҘј нҶөн•ҙ MLEлҘј лҚ” мһҳ мқҙн•ҙн•ҙліҙмһҗ

    • мЈјмӮ¬мң„мқҳ нҷ•лҘ  л¶„нҸ¬лҘј м•Ңкі  мӢ¶лӢӨ.
      • м җмҲҳ(likelihood)к°Җ к°ҖмһҘ лҶ’мқҖ кІғмқҖ $\theta_3$мқҙлӢӨ. л”°лқјм„ң мң„ 3к°ң мӨ‘м—җм„ңлҠ” лҚ°мқҙн„°лҘј мһҳ м„ӨлӘ…н•ҳлҠ” нҢҢлқјлҜён„°лқјкі  н•  мҲҳ мһҲмқҢ

Log-Likelihood

  • нҷ•лҘ  л¶„нҸ¬мқҳ к°ҖлҠҘм„ұмқ„ мёЎм •
  • м•һм—җм„ң мҶҢк°ңн–ҲлӢӨ мӢ¶мқҙ, LikelihoodлҠ” нҷ•лҘ  кіұмңјлЎң н‘ңнҳ„мқҙ лҗЁ
    • underflowмқҳ к°ҖлҠҘм„ұмқҙ мһҲмқҢ
  • л”°лқјм„ң LogлҘј м·Ён•ҳм—¬ кіұм…Ҳмқ„ лҚ§м…ҲмңјлЎң л°”кҫёкі , Log LikelihoodлЎң л¬ём ңлҘј н•ҙкІ°
    • лҚ§м…Ҳмқҙ кіұм…ҲліҙлӢӨ м—°мӮ°лҸ„ л№ лҰ„
  • $\prod_{i=1}^{n} P_{\theta}(\text{x} = x_{i})$ вҶ’ $\sum_{i=1}^{n} \log P_{\theta}(\text{x} = x_{i})$

MLE via Gradient Ascent

  • лһңлҚӨ мғқм„ұ лҢҖмӢ , Gradient AscentлҘј нҶөн•ҙ likelihood к°’мқ„ мөңлҢҖлЎң л§Ңл“ңлҠ” нҢҢлқјлҜён„°(рқңҪ)лҘј м°ҫмһҗ

    • $\theta \leftarrow \theta + \alpha \cdot \frac{\partial \mathcal{L}(\theta)}{\partial \theta}$

  • мҳҲм ң

    • n = 100лІҲ лҚҳмЎҢмқ„ л•Ң, k=27лІҲ м•һл©ҙ(True)мқҙ лӮҳмҳӨлҠ” лҸҷм „мқҙ мһҲлӢӨ. мқҙ лҸҷм „мқҳ нҷ•лҘ  л¶„нҸ¬(нҢҢлқјлҜён„° рқңҪ)лҘј м¶”м •н•ҳмһҗ.
      • Binomial Distribution
        • $\mathcal{L}(\theta) = \frac{n!}{k!(n-k)!} \theta^k (1-\theta)^{n-k}$

мҡ”м•Ҫ

  • мҡ°лҰ¬лҠ” к°ҖмғҒмқҳ нҷ•лҘ  л¶„нҸ¬лҘј лӘЁмӮ¬н•ҳлҠ” нҷ•лҘ  л¶„нҸ¬мқҳ нҢҢлқјлҜён„°(рқңҪ)лҘј м°ҫкі  мӢ¶лӢӨ
  • лӘ©н‘ң нҷ•лҘ  л¶„нҸ¬лЎңл¶Җн„° лҚ°мқҙн„°лҘј мҲҳм§‘н•ң нӣ„, лҚ°мқҙн„°лҘј мһҳ м„ӨлӘ…н•ҳлҠ” нҢҢлқјлҜён„°лҘј м°ҫмһҗ
    • LikelihoodлқјлҠ” к°’мқ„ нҶөн•ҙ м–јл§ҲлӮҳ мһҳ м„ӨлӘ…н•ҳлҠ”м§Җ м•Ң мҲҳ мһҲлӢӨ
    • Likelihood functionмқҖ рқңғ мқ„ мһ…л ҘмңјлЎң л°ӣм•„, лҚ°мқҙн„°л“Өмқҳ рқңғ м—җ лҢҖн•ң нҷ•лҘ  к°’мқҳ кіұмқ„ м¶ңл Ҙ
  • LikelihoodлҘј мөңлҢҖнҷ” н•ҳлҠ” нҢҢлқјлҜён„°лҘј м°ҫмңјл©ҙ, мЈјм–ҙм§„ лҚ°мқҙн„°лҘј к°ҖмһҘ мһҳ м„ӨлӘ…н•ңлӢӨ.
    • Gradient AscentлҘј нҶөн•ҙм„ң м°ҫмһҗ.
      • $\theta \leftarrow \theta + \alpha \cdot \frac{\partial \mathcal{L}(\theta)}{\partial \theta}$

2мһҘ. Probabilistic Perspective - Basic Statistics

кё°ліё нҶөкі„н•ҷ(Basic Statistics)

Random Variable & Probability Distribution

  • нҷ•лҘ  ліҖмҲҳ(Random Variable)лҠ” мӮ¬кұҙмқҳ мӢңн–үмқҳ кІ°кіј(нҷ•лҘ )лҘј н•ҳлӮҳмқҳ мҲҳм№ҳлЎң лҢҖмқ‘мӢңнӮ¬ л•Ңмқҳ нҷ•лҘ  к°’, мқјл°ҳм ҒмңјлЎң лҢҖл¬ёмһҗ XлҘј мӮ¬мҡ©
  • м–ҙл–Ө ліҖмҲҳ(Random Variable) xк°Җ рқ’ҷ лқјлҠ” к°’мқ„ к°Җм§Ҳ нҷ•лҘ 
    • $рқ‘ғ(\text{x}=рқ‘Ҙ) =рқ‘ғ(рқ‘Ҙ)$
  • нҷ•лҘ  л¶„нҸ¬ (н•ЁмҲҳ)
    • мһ…л Ҙ : нҷ•лҘ  ліҖмҲҳ x
    • м¶ңл Ҙ : xк°Җ к°Ғ к°’м—җ н•ҙлӢ№лҗ  л•Ңм—җ лҢҖн•ң нҷ•лҘ к°’
  • мқҙмӮ° нҷ•лҘ  ліҖмҲҳ
    • нҷ•лҘ ліҖмҲҳк°Җ м·Ён•  мҲҳ мһҲлҠ” к°’мқҳ мҲҳк°Җ мң н•ңн•ң ліҖмҲҳ
    • ex) лҸҷм „ лҚҳм§Җкё°, мЈјмӮ¬мң„ лҚҳм§Җкё°(X={0, 1, 2, 3})
  • м—°мҶҚ нҷ•лҘ  ліҖмҲҳ
    • нҷ•лҘ ліҖмҲҳк°Җ м·Ён•  мҲҳ мһҲлҠ” к°’мқҳ мҲҳк°Җ л¬ҙн•ңн•ң ліҖмҲҳ
    • ex) нӮӨ, лӘёл¬ҙкІҢ(P(50 вүӨ X вүӨ 60)) л“ұ

Function? or Value?

  • нҷ•лҘ к°’
    • $рқ‘ғ(рқ‘Ҙ) = P(\text{x}=x)$
  • нҷ•лҘ  л¶„нҸ¬ н•ЁмҲҳ
    • $P(\text{x})$
    • $P(y|x)$
      • $P(y|x) = P(\text{y}=y|\text{x}=x)$
      • Random Variable xк°Җ рқ‘Ҙ лқјлҠ” к°’мқ„ к°ҖмЎҢмқ„ л•Ң, Random Variable yк°Җ рқ‘Ұмқј нҷ•лҘ  к°’
    • $P(\text{y}|x)$
      • $P(\text{y}|x) = P(\text{y}|\text{x}=x)$
      • Random Variable xк°Җ рқ‘Ҙ лқјлҠ” к°’мқ„ к°ҖмЎҢмқ„ л•Ң, Random Variable yмқҳ л¶„нҸ¬
    • $P(y|\text{x})$
      • $P(y|\text{x})=f(\text{x})=P(\text{y}=y|\text{x})$
      • м–ҙл–Ө Random Variableмқҙ мһ…л ҘмңјлЎң мЈјм–ҙмЎҢмқ„ л•Ң, Random Variable yк°Җ рқ‘Ұмқј нҷ•лҘ  к°’мқ„ лұүм–ҙлӮҙлҠ” н•ЁмҲҳ

нҷ•лҘ  л¶„нҸ¬(Probability Distribution)

нҷ•лҘ  л¶„нҸ¬лҠ” мҲҳм№ҳлЎң лҢҖмқ‘лҗң нҷ•лҘ  ліҖмҲҳмқҳ к°ңлі„ к°’л“Өмқҙ к°Җм§ҖлҠ” нҷ•лҘ к°’мқҳ л¶„нҸ¬

  • мқҙмӮ° нҷ•лҘ  л¶„нҸ¬(Discrete Probability Distribution) : нҷ•лҘ  ліҖмҲҳк°Җ м·Ён•  мҲҳ мһҲлҠ” к°’мқҳ мҲҳк°Җ мң н•ңн•ң нҷ•лҘ  л¶„нҸ¬

    • нҷ•лҘ  м§Ҳлҹүн•ЁмҲҳ(Probability Mass Function) : нҷ•лҘ  ліҖмҲҳм—җм„ң нҠ№м • к°’м—җ лҢҖн•ң нҷ•лҘ мқ„ лӮҳнғҖлӮҙлҠ” н•ЁмҲҳ

      • нҷ•лҘ к°’мқҳ мҙқ н•©мқҖ 1
        • $\sum_{x} P(\text{x}=x) = 1, \text{ where } 0 \leq P(\text{x}=x) \leq 1, \forall x \in \chi$

    • мЈјмӮ¬мң„к°Җ мҳҲмӢңк°Җ лҗ  мҲҳ мһҲмқҢ(мқҙмӮ° нҷ•лҘ  л¶„нҸ¬)

  • м—°мҶҚ нҷ•лҘ  л¶„нҸ¬(Continuous Probability Distribution) : нҷ•лҘ  ліҖмҲҳк°Җ м·Ён•  мҲҳ мһҲлҠ” к°’мқҳ мҲҳк°Җ л¬ҙн•ңн•ң нҷ•лҘ  л¶„нҸ¬

    • нҷ•лҘ  л°ҖлҸ„ н•ЁмҲҳ(robability Density Function , PDF) : нҷ•лҘ  ліҖмҲҳмқҳ л¶„нҸ¬лҘј лӮҳнғҖлӮҙлҠ” н•ЁмҲҳ

      • л©ҙм Ғмқҳ н•©мқҙ 1
        • $\int P(x) , dx = 1, \text{ where } P(x) \geq 0, \forall x \in \mathbb{R}$
      • н•ЁмҲҳк°’мқҙ 1ліҙлӢӨ нҒҙ мҲҳ мһҲмқҢ

      вҖ» мЈјмӮ¬мң„лҠ” нҷ•лҘ м§Ҳлҹүн•ЁмҲҳ(мқҙмӮ°), нҷ•лҘ л°ҖлҸ„н•ЁмҲҳлҠ” м—°мҶҚн•ЁмҲҳ

    • м—°мҶҚ нҷ•лҘ  ліҖмҲҳмқҳ кІҪмҡ°, м–ҙл–Ө мғҳн”Ңмқҙ мЈјм–ҙмЎҢмқ„ л•Ң, нҷ•лҘ к°’мқ„ м•Ң мҲҳ м—ҶлӢӨ. (лҶ’мқҙлҠ” лӢЁм§Җ нҷ•лҘ  л°ҖлҸ„)

  • кІ°н•© нҷ•лҘ  л¶„нҸ¬(Joint Probability Distribution)

    • л‘җ к°ң мқҙмғҒмқҳ Random Variableмқҙ кІ°н•©лҗҳм—Ҳмқ„ л•Ңмқҳ нҷ•лҘ  л¶„нҸ¬

    • л‘җ мқҙлІӨнҠё AмҷҖ Bк°Җ лҸҷмӢңм—җ л°ңмғқн•  нҷ•лҘ мқ„ мқҳлҜён•©лӢҲлӢӨ. мқҙлҘј кё°нҳёлЎң лӮҳнғҖлӮҙл©ҙ P(A вҲ© B) лҳҗлҠ” P(A, B)лЎң н‘ңнҳ„

    • P(A, B) = P(A|B) * P(B) = P(B|A) * P(A)

      • P(A, B) : мӮ¬кұҙ AмҷҖ мӮ¬кұҙ Bк°Җ лҸҷмӢңм—җ л°ңмғқн•  нҷ•лҘ 
      • P(A|B) : мӮ¬кұҙ Bк°Җ л°ңмғқн•ң мЎ°кұҙ н•ҳм—җм„ң мӮ¬кұҙ Aк°Җ л°ңмғқн•  нҷ•лҘ 
      • P(B) : мӮ¬кұҙ Bк°Җ л°ңмғқн•  нҷ•лҘ 

мЎ°кұҙл¶Җ нҷ•лҘ (Conditional Probability)

  • м–ҙл–Ө мӮ¬кұҙ Aк°Җ мқјм–ҙлӮ¬мқ„ л•Ң мӮ¬кұҙ Bк°Җ мқјм–ҙлӮ  нҷ•лҘ мқ„ мқҳлҜё. мҰү, мӮ¬кұҙ Aк°Җ мЈјм–ҙмЎҢмқ„ л•Ңмқҳ мӮ¬кұҙ Bмқҳ нҷ•лҘ 

  • мЎ°кұҙл¶Җ нҷ•лҘ  л¶„нҸ¬

    • лһңлҚӨ ліҖмҲҳмқҳ л¶„нҸ¬к°Җ лӢӨлҘё лһңлҚӨ ліҖмҲҳмқҳ к°’м—җ мқҳн•ҙ м–ҙл–»кІҢ мҳҒн–Ҙмқ„ л°ӣлҠ”м§ҖлҘј м„ӨлӘ…н•ҳлҠ” нҷ•лҘ  л¶„нҸ¬

    • мҰү, н•ҳлӮҳмқҳ лһңлҚӨ ліҖмҲҳ Xмқҳ к°’мқҙ мЈјм–ҙмЎҢмқ„ л•Ң, лӢӨлҘё лһңлҚӨ ліҖмҲҳ Yмқҳ л¶„нҸ¬лҘј мқҳлҜё

    • $P(\text{y}|\text{x}) = \frac{P(\text{x}, \text{y})}{P(\text{x})}$ (xк°Җ мЎ°кұҙмңјлЎң мЈјм–ҙмЎҢмқ„ л•Ң yмқҳ нҷ•лҘ к°’)

      P y given x лқјкі  мқҪмқҢ

    • мЎ°кёҲлҚ” м№ңн•ҙм ём•јн•  нҳ•нғң(кІ°н•©нҷ•лҘ л¶„нҸ¬ нҳ•нғң)

      • $P(\text{x}, \text{y}) = P(\text{y} ,|, \text{x})P(\text{x})$

    • мҳҲлҘј л“Өм–ҙ, лһңлҚӨ ліҖмҲҳ Xк°Җ вҖҳмҳӨлҠҳ л№„к°Җ мҳ¬ нҷ•лҘ вҖҷмқ„ лӮҳнғҖлӮёлӢӨкі  н•  л•Ң, лһңлҚӨ ліҖмҲҳ YлҠ” вҖҳл№„к°Җ мҳ¬ л•Ң мҡ°мӮ°мқ„ к°Җм§Җкі  к°Ҳ нҷ•лҘ вҖҷмқ„ лӮҳнғҖлӮј мҲҳ мһҲмқҢ. мқҙ кІҪмҡ°, вҖҳмҡ°мӮ°мқ„ к°Җм§Җкі  к°Ҳ нҷ•лҘ вҖҷмқҖ вҖҳл№„к°Җ мҳ¬ нҷ•лҘ вҖҷм—җ мқҳн•ҙ мҳҒн–Ҙмқ„ л°ӣмқҢ. мқҙлҹ° мғҒнҷ©мқ„ лӘЁлҚёл§Ғн•ң кІғмқҙ мЎ°кұҙл¶Җ нҷ•лҘ  л¶„нҸ¬мқҙлӢӨ.

лІ мқҙмҰҲ м •лҰ¬(Bayes Theorm)

  • лІ мқҙмҰҲ м •лҰ¬лҠ” нҷ•лҘ  мқҙлЎ кіј нҶөкі„н•ҷм—җм„ң көүмһҘнһҲ мӨ‘мҡ”н•ң к°ңл…җмңјлЎң, мЎ°кұҙл¶Җ нҷ•лҘ м—җ кҙҖн•ң м •лҰ¬

  • мЎ°кұҙл¶Җ нҷ•лҘ мқҳ к°ңл…җмқ„ нҷ•мһҘн•ҳкі  мқҙлҘј л’Өм§‘лҠ” лҚ° мӮ¬мҡ©

  • мЈјм–ҙм§„ мӮ¬кұҙ Bк°Җ л°ңмғқн–Ҳмқ„ л•Ң, мӮ¬кұҙ Aк°Җ л°ңмғқн•  нҷ•лҘ мқ„ м•Ңм•„лӮҙлҠ” л°©лІ•мқ„ м ңкіө. мҰү, мқҙлҜё м•Ңкі  мһҲлҠ” м •ліҙм—җ кё°л°ҳн•ҳм—¬ мғҲлЎңмҡҙ мӮ¬кұҙмқҳ нҷ•лҘ мқ„ м¶”м •н•ҳлҠ” л°©лІ•

  • $P(A ,|, B) = \frac{P(B,|,A)P(A)}{P(B)}$,

    • (A|B)лҠ” мӮ¬кұҙ Bк°Җ л°ңмғқн–Ҳмқ„ л•Ң мӮ¬кұҙ Aк°Җ л°ңмғқн•  мЎ°кұҙл¶Җ нҷ•лҘ мқ„ мқҳлҜён•ҳл©°, мқҙлҘј мӮ¬нӣ„ нҷ•лҘ (Posterior probability)мқҙлқјкі  н•Ё
    • P(B|A)лҠ” мӮ¬кұҙ Aк°Җ л°ңмғқн–Ҳмқ„ л•Ң мӮ¬кұҙ Bк°Җ л°ңмғқн•  мЎ°кұҙл¶Җ нҷ•лҘ мқ„ мқҳлҜён•ҳл©°, мқҙлҘј к°ҖлҠҘлҸ„(Likelihood)лқјкі  н•Ё
    • P(A)лҠ” мӮ¬кұҙ Aк°Җ л°ңмғқн•  нҷ•лҘ мқ„ мқҳлҜён•ҳл©°, мқҙлҘј мӮ¬м „ нҷ•лҘ (Prior probability)лқјкі  н•Ё
    • P(B)лҠ” мӮ¬кұҙ Bк°Җ л°ңмғқн•  нҷ•лҘ мқ„ мқҳлҜён•ҳл©°, мқҙлҘј мҰқкұ°(Evidence)лқјкі  н•Ё

    AлҘј вҖңмҳӨлҠҳ л№„к°Җ мҳ¬ кІғвҖқмқҙлқјлҠ” мӮ¬кұҙ, BлҘј вҖңмҳӨлҠҳ н•ҳлҠҳмқҙ нқҗлҰјвҖқмқҙлқјлҠ” мӮ¬кұҙмқҙлқјкі  к°Җм •н•©мӢңлӢӨ. мқҙ кІҪмҡ°, лІ мқҙмҰҲ м •лҰ¬лҠ” лӢӨмқҢкіј к°ҷмқҙ н‘ңнҳ„лҗ  мҲҳ мһҲмҠөлӢҲлӢӨ.

    • мӮ¬нӣ„ нҷ•лҘ (P(A|B)), мҰү вҖңн•ҳлҠҳмқҙ нқҗлҰҙ л•Ң л№„к°Җ мҳ¬ нҷ•лҘ вҖқмқҖ м•Ңкі  мӢ¶мқҖ мөңмў…м Ғмқё нҷ•лҘ мһ…лӢҲлӢӨ. мқҙлҠ” мҡ°лҰ¬к°Җ мқҙлҜё м•Ңкі  мһҲлҠ” вҖңн•ҳлҠҳмқҙ нқҗлҰјвҖқмқҙлқјлҠ” м •ліҙлҘј л°”нғ•мңјлЎң вҖңмҳӨлҠҳ л№„к°Җ мҳ¬ кІғвҖқмқҙлқјлҠ” мғҲлЎңмҡҙ мӮ¬кұҙм—җ лҢҖн•ң нҷ•лҘ мқ„ м—…лҚ°мқҙнҠён•ҳлҠ” кІғмқ„ мқҳлҜён•©лӢҲлӢӨ.
    • к°ҖлҠҘлҸ„(P(B|A)), мҰү вҖңл№„к°Җ мҳ¬ л•Ң н•ҳлҠҳмқҙ нқҗлҰҙ нҷ•лҘ вҖқмқҖ мҡ°лҰ¬к°Җ мқҙлҜё м•Ңкі  мһҲлҠ” вҖңл№„к°Җ мҳ¬ кІғвҖқмқҙлқјлҠ” мӮ¬кұҙмқҙ мЈјм–ҙмЎҢмқ„ л•Ң вҖңн•ҳлҠҳмқҙ нқҗлҰјвҖқмқҙлқјлҠ” мӮ¬кұҙмқҳ нҷ•лҘ мһ…лӢҲлӢӨ. мқҙлҠ” мӮ¬кұҙ Aк°Җ мЈјм–ҙмЎҢмқ„ л•Ң мӮ¬кұҙ Bк°Җ л°ңмғқн•  нҷ•лҘ мқ„ лӮҳнғҖлғ…лӢҲлӢӨ.

  • $P(h ,|, D) = \frac{P(D,|,h)P(h)}{P(D)}$

    • $P(h ,|, D)P(D) = P(D,|,h)P(h) =P(h,D)$

    • к°Җм„Өмқҙ мһҲмқ„ л•Ң лҚ°мқҙн„°к°Җ мһҲмқ„ нҷ•лҘ мқ„ л’Өм§‘м–ҙм„ң м•Ң мҲҳ мһҲкІҢ н•ҙмӨҖлӢӨ.

мЎ°кұҙл¶Җ нҷ•лҘ кіј лІ мқҙмҰҲ м •лҰ¬мқҳ м°ЁмқҙлҘј лӢӨмқҢмқҳ мҳҲмӢңлЎң м•Ңм•„ліҙмһҗмҳҲмӢң : вҖңл№„к°Җ мҳӨлҠ” лӮ м—җ мҡ°мӮ°мқ„ л“Өкі  лӮҳк°ҖлҠ” нҷ•лҘ вҖқ

мЎ°кұҙл¶Җ нҷ•лҘ  вҖҳл№„к°Җ мҳӨлҠ” лӮ вҖҷмқҙлқјлҠ” мӮ¬кұҙмқҙ мқҙлҜё мқјм–ҙлӮ¬мқ„ л•Ң, к·ё мЎ°кұҙ н•ҳм—җм„ң вҖҳмҡ°мӮ°мқ„ л“Өкі  лӮҳк°ҖлҠ”вҖҷ мӮ¬кұҙмқҙ мқјм–ҙлӮ  нҷ•лҘ мқ„ кі„мӮ°н•ҳлҠ” кІғмқҙ мЎ°кұҙл¶Җ нҷ•лҘ 

лІ мқҙмҰҲ м •лҰ¬ вҖқмҡ°мӮ°мқ„ л“Өкі  лӮҳк°ҖлҠ” мӮ¬лһҢ мӨ‘м—җм„ң л№„к°Җ мҳӨлҠ” кІҪмҡ°вҖқ

мЈјліҖ л¶„нҸ¬(Marginal Distribution)

  • м—¬лҹ¬ ліҖмҲҳмқҳ кІ°н•© нҷ•лҘ  л¶„нҸ¬м—җм„ң н•ң к°ң лҳҗлҠ” мқјл¶Җ ліҖмҲҳмқҳ л¶„нҸ¬лҘј лӮҳнғҖлғ„, мқҙлҠ” вҖҳмЈјліҖнҷ”(marginalization)вҖҷлқјлҠ” кіјм •мқ„ нҶөн•ҙ м–»м–ҙм§җ

  • мЈјліҖнҷ”лҠ” кІ°н•© нҷ•лҘ  л¶„нҸ¬м—җм„ң кҙҖмӢ¬ мһҲлҠ” ліҖмҲҳлҘј м ңмҷён•ң лӮҳлЁём§Җ ліҖмҲҳл“Өмқ„ лӘЁл‘җ н•©мӮ°(мқҙмӮ°)н•ҳкұ°лӮҳ м Ғл¶„(м—°мҶҚ)н•ҳлҠ” кіјм •мқ„ мқҳлҜё

  • мЈјліҖ нҷ•лҘ  л¶„нҸ¬лҘј нҶөн•ҙ, нҠ№м • ліҖмҲҳк°Җ лӢӨлҘё ліҖмҲҳл“ӨлЎңл¶Җн„° лҸ…лҰҪм ҒмңјлЎң м–ҙл–»кІҢ л¶„нҸ¬н•ҳлҠ”м§ҖлҘј нҢҢм•…н•  мҲҳ мһҲмқҢ, мқҙлҘј нҶөн•ҙ к·ё ліҖмҲҳмқҳ л¶„нҸ¬лҘј лӢЁлҸ…мңјлЎң ліј мҲҳ мһҲкІҢ лҗЁ

  • мЈјліҖ л¶„нҸ¬лҠ” нҷ•лҘ лЎ мқҳ н•өмӢ¬ к°ңл…җмқё кІ°н•© нҷ•лҘ , мЎ°кұҙл¶Җ нҷ•лҘ , к·ёлҰ¬кі  мқҙл“Өмқҳ кҙҖкі„лҘј м—°кІ°н•ҙ ліҙм—¬мЈјл©°, мЈјм–ҙм§„ лҚ°мқҙн„°м—җм„ң нҠ№м • ліҖмҲҳмқҳ нҷ•лҘ  л¶„нҸ¬лҘј лҸ…лҰҪм ҒмңјлЎң нҢҢм•…н•ҳлҠ” лҚ° мң мҡ©

  • Marginal Distributionмқҳ кіөмӢқ

    • $P(x) = \int P(x,z) dz$ ( P(x, z)лҠ” xмҷҖ zмқҳ кІ°н•© нҷ•лҘ  л¶„нҸ¬лҘј лӮҳнғҖлӮҙкі , мқҙлҘј zм—җ лҢҖн•ҙ м Ғл¶„н•ҳл©ҙ xмқҳ мЈјліҖ нҷ•лҘ  л¶„нҸ¬ xлҘј м–»кІҢ лҗЁ)

      $= \int P(x|z)P(z) dz$ (кІ°н•© нҷ•лҘ л¶„нҸ¬лҘј мЎ°кұҙл¶Җ нҷ•лҘ  P(x | z)мҷҖ zмқҳ мЈјліҖ нҷ•лҘ  P(z)мқҳ кіұмңјлЎң л¶„н•ҙн•ҳкі  мқҙлҘј zм—җ лҢҖн•ҙ м Ғл¶„н•ҳм—¬ xмқҳ мЈјліҖнҷ•лҘ  л¶„нҸ¬ P(x)лҘј м–»кІҢлҗЁ)

      $= \int P(z|x)P(x) dz = P(x)\int P(z|x) dz = P(x)$ $(вҲө\int P(z|x) dz=1)$ (мЎ°кұҙл¶Җ нҷ•лҘ  $P(z | x)$к°Җ zм—җ лҢҖн•ҙ м Ғл¶„н•ҳл©ҙ 1мқҙ лҗЁ, мҷңлғҗн•ҳл©ҙ мЎ°кұҙл¶Җ нҷ•лҘ  P(z|x)лҠ” zмқҳ нҷ•лҘ  л¶„нҸ¬лҘј лӮҳнғҖлӮҙкё° л•Ңл¬ё)

Expectation and Sampling

  • кё°лҢҖк°’(expectation)мқҖ нҷ•лҘ  л¶„нҸ¬мҷҖ к·ём—җ лҢҖмқ‘н•ҳлҠ” н•ЁмҲҳмқҳ к°ҖмӨ‘нҸүк· мқ„ мқҳлҜё

  • кё°лҢҖк°’мқҳ мҲҳмӢқ (м•һмңјлЎң мһҗмЈј ліҙкІҢлҗ  мҲҳмӢқ)

    • $\mathbb{E}

      {\text{x} \sim P(\text{x})}[f(x)] = \sum

      {x \in \chi} P(x) \cdot f(x)$

      • $\chi$лҠ” лӘЁл“  к°ҖлҠҘн•ң $x$мқҳ м§‘н•©, $f(x)$лҠ” $x$м—җ лҢҖн•ң н•ЁмҲҳ, $P(x)$лҠ” $x$мқҳ нҷ•лҘ  л¶„нҸ¬

  • f(x)лқјлҠ” м–ҙл–Ө н•ЁмҲҳк°Җ мһҲлҠ”лҚ°, P(x)м—җм„ң мғҳн”Ңл§Ғн•ң xлҘј н•ЁмҲҳм—җ л„Јм–ҙлҙ„

  • мЈјмӮ¬мң„мқҳ кё°лҢҖк°’мқҖ?

    • $\mathbb{E}{x \sim P(x)}[f(x)] = \sum{x \in {1,2,3,4,5,6}} P(x = x) \cdot f(x)$

      $=\frac{1}{6}Г—(f(1)+f(2)+f(3)+f(4)+f(5)+f(6))$

      $= \frac{1}{6}Г—(1+2+3+4+5+6)=3.5 \text{, where } f(x)=x$

    cf) лӘЁл“  кё°лҢҖк°’мқҙ к°ҷмңјл©ҙ uniform distribution л”°лқјм„ң мӢңк·ёл§Ҳ 1/n *f(x)

  • $P(x) = \int P(x,z) dz$

    $= \int P(x|z)P(z) dz$

    $= \mathbb{E}_{\text{z} \sim P(\text{z})}[P(x|\text{z})]$ (P(x)лҘј zм—җ лҢҖн•ң P(x|z)мқҳ кё°лҢҖк°’мңјлЎң н‘ңнҳ„)

Monte-Carlo л°©лІ•

  • л¬ҙмһ‘мң„ н‘ңліёмқ„ мӮ¬мҡ©н•ҳм—¬ мҲҳн•ҷм Ғ л¬ём ңлҘј н•ҙкІ°н•ҳлҠ” нҶөкі„м Ғмқё л°©лІ•
  • мЈјлЎң лӢӨмқҢкіј к°ҷмқҖ мғҒнҷ©м—җм„ң мӮ¬мҡ©лҗЁ
    • к°•нҷ” н•ҷмҠөм—җм„ңлҠ” Monte Carlo л°©лІ•мқ„ мӮ¬мҡ©н•ҳм—¬ м—җмқҙм „нҠёмқҳ м •мұ…(policy)мқ„ нҸүк°Җн•ҳкұ°лӮҳ м—…лҚ°мқҙнҠё
    • л”Ҙлҹ¬лӢқм—җм„ң ліөмһЎн•ң нҷ•лҘ  лӘЁлҚём—җм„ң мғҳн”Ңмқ„ м¶”м¶ңн•ҳкё° мң„н•ҙ Monte Carlo мғҳн”Ңл§Ғ кё°лІ•мқ„ мӮ¬мҡ©

One-hot Encoding

Categorical Value vs Continuous Value

  • Categorical Value
    • ліҙнҶөмқҖ discrete value
    • лӢЁм–ҙ, нҒҙлһҳмҠӨ
  • Continuous Value
    • нӮӨ, лӘёл¬ҙкІҢ
  • Categorical ValueмҷҖ Continuous Valueмқҳ к°ҖмһҘ кІ°м •м Ғмқё м°Ёмқҙм җ
    • Continous valueлҠ” л№„мҠ·н•ң к°’мқҖ л№„мҠ·н•ң мқҳлҜёлҘј м§ҖлӢҲм§Җл§Ң
    • Categorical valueлҠ” л№„мҠ·н•ң к°’мқјм§ҖлқјлҸ„ мғҒкҙҖм—ҶлҠ” мқҳлҜёлҘј м§ҖлӢҢлӢӨ.

One-hot Encodingмқҳ н•„мҡ”м„ұ

  • One-hot Encodingмқҳ н•„мҡ”м„ұмқ„ лҠҗлҒјкё° мң„н•ҙ м•„лһҳмқҳ лӢЁм–ҙлҘј мӮ¬м „ мҲңмңјлЎң indexм—җ mapping н•ҙліҙмһҗ

    0 1 2 3 4 5 6
    кіөмұ… л”ұн’Җ ліјнҺң мғӨн”„ м—°н•„ мһҗ н•„кё°мһҘ
    • мғҒмӢқм ҒмңјлЎңлҠ”
      • distance(м—°н•„, ліјнҺң) < distance(м—°н•„, мһҗ)
      • distance(кіөмұ…, н•„кё°мһҘ) < distance(кіөмұ…, л”ұн’Җ)
    • н•ҳм§Җл§Ң мқҙ н…Ңмқҙлё”м—җм„ңлҠ” м•„лһҳмҷҖ к°ҷмқҖ кІ°кіјк°Җ лӮҳмҳЁлӢӨ.
      • |м—°н•„ - ліјнҺң| = 2 > 1 = |м—°н•„ - мһҗ|
      • |кіөмұ… - н•„кё°мһҘ| = 6 > 1 = |кіөмұ… - л”ұн’Җ|
    • мҰү, лІ”мЈјнҳ• лҚ°мқҙн„°лҘј мһ„мқҳмқҳ мҲ«мһҗлЎң н‘ңнҳ„н•ҳкІҢ лҗҳл©ҙ, н…ҚмҠӨнҠё к°„м—җ мӣҗлһҳ мЎҙмһ¬н•ҳм§Җ м•Ҡм•ҳлҚҳ вҖҳнҒ¬кё°вҖҷлӮҳ вҖҳмҲңм„ңвҖҷмқҳ к°ңл…җмқҙ л¶Җм—¬лҗЁ. мқҙлЎң мқён•ҙ лІ”мЈјнҳ• лҚ°мқҙн„° к°„м—җ мӢӨм ңлЎңлҠ” м—ҶлҠ” кұ°лҰ¬ нҳ№мқҖ м°Ёмқҙк°Җ мЎҙмһ¬н•ҳлҠ” кІғмІҳлҹј н•ҙм„қлҗҳм–ҙ, л¶Ҳн•„мҡ”н•ҳкұ°лӮҳ мһҳлӘ»лҗң м •ліҙлҘј н•ҷмҠөн•ҳлҠ” кІ°кіјлҘј мҙҲлһҳн•  мҲҳ мһҲмқҢ

    • лІ”мЈј к°„мқҳ мҲңм„ң м—ҶмқҢ
    • нҠ№м„ұ к°„мқҳ лҸ…лҰҪм„ұ ліҙмһҘ

  • One-hot Encodingмқҳ лӢЁм җ

    • м„ңлЎң лӢӨлҘё л‘җ лІЎн„°лҠ” н•ӯмғҒ м§Ғкөҗ(orthogonal)н•ңлӢӨ. (element-wise кіұ = 0)
      • Cosine Similarityк°Җ 0, л”°лқјм„ң л‘җ мғҳн”Ң мӮ¬мқҙмқҳ мң мӮ¬лҸ„(кұ°лҰ¬)лҘј кө¬н•  мҲҳ м—ҶмқҢ

Embedding Vectors

Motivation of Embedding Vectors

  • NLPм—җм„ң лӢЁм–ҙлҠ” categorical value & discrete valueмқҳ мҶҚм„ұмқ„ к°–мқҢ

    • л”°лқјм„ң one-hot representationмңјлЎң н‘ңнҳ„
    • н•ҳм§Җл§Ң мқҙлҠ” мӢӨм ң мЎҙмһ¬н•ҳлҠ” лӢЁм–ҙ мӮ¬мқҙмқҳ мң мӮ¬лҸ„лҘј н‘ңнҳ„н•  мҲҳ м—ҶмқҢ
    • л”°лқјм„ң мӢӨм ңм ҒмңјлЎңлҠ” мўҖлҚ” кі кёүнҷ”лҗң Embedding кё°лІ•мқ„ мӮ¬мҡ©

  • лӢӨлҘё Embedding Vectors н‘ңнҳ„ кё°лІ•

    • Contextual Word Embedding (л¬ёл§Ҙм Ғ лӢЁм–ҙ мһ„лІ л”©)

      • кё°мЎҙмқҳ лӢЁм–ҙ мһ„лІ л”© л°©лІ•мқҖ лӢЁм–ҙмқҳ мқҳлҜёк°Җ л¬ёл§Ҙм—җ л”°лқј лӢ¬лқјм§Ҳ мҲҳ мһҲлӢӨлҠ” м җмқ„ кі л Өн•ҳм§Җ лӘ»н–ҲлҠ”лҚ° мқҙлҘј н•ҙкІ°н•ҳкё° мң„н•ҙ л“ұмһҘ

      • лӢЁм–ҙлҘј мһ„лІ л”©н•  л•Ң мЈјліҖ л¬ёл§Ҙмқ„ кі л Өн•ҙ лҸҷмқјн•ң лӢЁм–ҙлқјлҸ„ лӢӨлҘё л¬ёл§Ҙм—җм„ңлҠ” лӢӨлҘё мһ„лІ л”© лІЎн„°лҘј к°–кІҢн•Ё

      • BERT, ELMO л“ұмқҙ н•ҙлӢ№лҗЁ

      • мҪ”л“ңлЎң кө¬нҳ„н•ҳкё°

    • Subword Embedding (м„ңлёҢмӣҢл“ң мһ„лІ л”©)

      • нҠ№м • м–ём–ҙл“ӨмқҖ лӢЁм–ҙлҘј лҚ” мһ‘мқҖ мқҳлҜё лӢЁмң„лЎң л¶„лҰ¬н•  мҲҳ мһҲмқҢ
      • мқҙлҹ° кІҪмҡ°, м„ңлёҢмӣҢл“ң мһ„лІ л”©мқ„ мӮ¬мҡ©н•ҳм—¬ мһ‘мқҖ лӢЁмң„л“Өмқ„ н•ҷмҠөн•  мҲҳ мһҲмқҢ
      • BPE(Byte Pair Encoding), SentencePiece л“ұмқҙ н•ҙлӢ№лҗЁ

    • Document Embedding (л¬ём„ң мһ„лІ л”©)

      • л¬ём„ң м „мІҙлҘј н•ҳлӮҳмқҳ лІЎн„°лЎң н‘ңнҳ„н•ҳлҠ” л°©лІ•
      • л¬ём„ң мһ„лІ л”©мқҖ л¬ём„ңмқҳ м „мІҙм Ғмқё мқҳлҜёлҘј мқҙн•ҙн•ҳлҠ”лҚ° лҸ„мӣҖмқҙ лҗЁ
      • Doc2Vec, FastText л“ұмқҙ н•ҙлӢ№

    • Word Embedding (лӢЁм–ҙ мһ„лІ л”©)

      • кі м°Ёмӣҗмқҳ One-hot лІЎн„°лҘј м Җм°Ёмӣҗмқҳ мӢӨмҲҳ лІЎн„°лЎң ліҖнҷҳн•ҳлҠ” кё°лІ•
      • л№„мҠ·н•ң мқҳлҜёлҘј к°Җм§„ лӢЁм–ҙл“Өмқҙ лІЎн„° кіөк°„м—җм„ң к°Җк№Ңмқҙ мң„м№ҳн•ҳлҸ„лЎқ н•ҷмҠө
      • Word2Vec, GloVe л“ұмқҙ н•ҙлӢ№лҗЁ

  • к°ңмқём Ғмқё кІҪн—ҳмңјлЎңлҠ” Contextual Word Embedding + Subword Embeddingмқ„ кІ°н•©н•ҙм„ң к°ҖмһҘ л§Һмқҙ мӮ¬мҡ©н•ҳлҠ” кІғ к°ҷмңјл©°, Word EmbeddingмқҖ к№Җкё°нҳ„ лӢҳмқҳ л§җм”Җм—җ мқҳн•ҳл©ҙ Embeddingм—җ м Ғн•©н•ҳм§Җ м•ҠлӢӨкі  л§җм”Җн•ҙмЈјмӢ  кІғмңјлЎң м•Ңкі  мһҲлӢӨ.

мҡ”м•Ҫ

  • Categorical ValueлҠ” One-hot Encodingмқ„ нҶөн•ҙ лІЎн„°лЎң н‘ңнҳ„лҗЁ
  • Sparse VectorлҠ” лІЎн„° к°„ мң мӮ¬лҸ„ кі„мӮ°мқҙ м–ҙл ӨмӣҖ вҶ’ One-hotмқҳ лӢЁм җ

Learning Rate

Learning Rate in Gradient Descent

  • нҢҢлқјлҜён„°к°Җ м—…лҚ°мқҙнҠё лҗ  л•Ң, gradientмқҳ нҒ¬кё°м—җ мҳҒн–Ҙмқ„ л°ӣкІҢ лҗЁ
    • мқҙ л•Ң, learning rateк°Җ step-sizeлҘј м •н•ҙмЈјкІҢ лҗЁ
  • Equation
    • $\theta \gets \theta - \eta \frac{\partial L(\theta)}{\partial \theta} = \theta - \eta \nabla_\theta L(\theta)$

Learning Rate м—җ л”°лҘё мөңм Ғнҷ” (лҚ°мқҙн„°лӮҳ лӘЁлҚё м•„нӮӨн…ҚмІҳм—җ л”°лқј lrмқҖ ліҖн•Ё)

  • Large LR
    • л„Ҳл¬ҙ нҒ° Lossк°Җ л°ңмӮ°н•  мҲҳ мһҲмқҢ
  • Small LR
    • л„Ҳл¬ҙ мһ‘мқҖ LRмқҖ мҲҳл ҙмқҙ лҠҰмқҢ
    • мһҗм№« local minimaм—җ л№ м§Ҳ мҲҳ мһҲмқҢ

    • кі лҜјн•  л°”м—җ к·ёлғҘ м•„мЈј мһ‘мқҖ к°’(eg. 1e-4)мңјлЎң мҳӨлһҳ лҸҢл ӨлҸ„ кҙңм°®мқҢ

7мһҘ Wrap up

Why we do gradient descent?

  • мӢӨмһ¬н•ҳм§Җл§Ң м•Ң мҲҳ м—ҶлҠ” н•ЁмҲҳ $f^*$лҘј к·јмӮ¬н•ҳкі  мӢ¶мқҢ
  • лӮҳмқҳ лӘЁлҚё(н•ЁмҲҳ)мқҳ $f_\theta$ нҢҢлқјлҜён„° рқңҪлҘј мЎ°м Ҳ
